# Classification Review

## Source-scope decisions

1. Non-UGC RPSC recruitment scans have been removed from the UGC NET source collection and are not part of any question count.
2. No local PDF is currently verifiable as a Rajasthan SET Computer Science question paper. Rajasthan SET coverage is therefore absent because source papers are missing.

## Unit-boundary rules for manual review

- Boolean-function simplification belongs primarily to Unit 1; gates, flip-flops, multiplexers and circuit implementation belong to Unit 2.
- Graph definitions and structural properties belong to Unit 1. Shortest paths, tree traversals and spanning trees are explicitly named in Unit 1 and therefore remain Unit 1 when they are the tested graph-theory concept; implementation, complexity or comparative algorithm analysis receives a Unit 7 cross-classification.
- Mathematical induction belongs to Unit 1; algorithmic recurrence solving and asymptotic analysis belong to Unit 7.
- Propositional/predicate logic belongs to Unit 1; logic used specifically as an AI knowledge-representation formalism may belong to Unit 10.
- PERT/CPM belongs to Unit 1 Optimization; software project estimation questions without network scheduling belong to Unit 6.

Any question whose primary learning objective remains unclear after applying these rules must be marked `manual-review` rather than guessed.
