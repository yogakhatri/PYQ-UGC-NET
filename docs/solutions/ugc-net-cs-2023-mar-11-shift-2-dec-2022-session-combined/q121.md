# Question 121

Match List I with List II LIST I LIST II System Requirements Design. A. Product complexity Software design. Structured System analysis II. Validation techniques. Coupling & Cohesion V. Sofiware cost estimation. D. Symbolic Execution Choose the correct answer from the options given below:

- **1.** A-II, B-I, C-IV, D-I
- **2.** A-III, B-I, C-IV, D-II
- **3.** A-IV, B-I, C-II, D-III
- **4.** A-III, B-IV, C-1, D-II

## Answer

**3. A-IV, B-I, C-II, D-III**

## Detailed solution

Product complexity is a software cost-estimation factor (A–IV). Structured systems analysis develops system requirements/design (B–I). Coupling and cohesion are software-design measures (C–II). Symbolic execution is a validation technique (D–III).

## Why the other options do not work

Every other mapping assigns at least one concept to the wrong development activity.

## Key idea to remember

Coupling/cohesion assess design; symbolic execution validates program paths.
