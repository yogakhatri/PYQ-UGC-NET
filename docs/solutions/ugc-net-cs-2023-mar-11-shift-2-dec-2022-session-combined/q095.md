# Question 95

Stoption 1D-394751 SEND 187045 Which of the following statement is/are false? A. The processor has direct access to both primary and secondary memory B. Primary memory stores the active instructions and data for the program being executed on the process Secondary memory is used as a backup memory Memory system is implemented on a single level memory Choose the correct answer from the options given below:

- **1.** A and B only
- **2.** B and C only
- **3.** A, B and C only
- **4.** A and D Only

## Answer

**4. A and D Only**

## Detailed solution

A is false because the processor directly addresses registers/cache/main memory, while secondary storage is accessed through I/O. B and C are true in the intended hierarchy. D is false because practical memory systems use multiple levels. Therefore A and D are false.

## Why the other options do not work

Options 1–3 incorrectly mark B or C false or fail to include the false single-level claim D.

## Key idea to remember

The memory hierarchy is multi-level, and CPU load/store instructions do not directly address disk blocks.
