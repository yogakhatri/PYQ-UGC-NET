# Question 72

No.2 BID:18702 COCOMO stands for

- **1.** Consumed cost model
- **2.** Constructive cost model
- **3.** Common control model
- **4.** Composition cost model

## Answer

**2. Constructive cost model**

## Detailed solution

COCOMO is the Constructive Cost Model, an empirical software cost-estimation model originally developed by Barry Boehm. It estimates development effort, schedule, and cost from software size and project-specific cost drivers.

## Why the other options do not work

Consumed cost model, common control model, and composition cost model are not the expansion of the COCOMO acronym.

## Key idea to remember

COCOMO = Constructive Cost Model; remember it as a software effort and schedule estimation model.
