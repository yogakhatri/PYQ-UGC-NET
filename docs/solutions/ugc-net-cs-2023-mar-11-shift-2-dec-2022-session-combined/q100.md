# Question 100

дорон 10-3201 S. D: 87050 Identify the correct operation which produces the below given output based on two relations R1 and R2. Name Class al 9 b1 no 11 dl R2 Class Name 5 fl 8 Output: Name Class bl 5 Choose the correct answer from the options below:

- **1.** RI NR2
- **2.** RI - R2
- **3.** RI - (RI- R2)
- **4.** R1 U R2

## Answer

**1. RI NR2**

## Detailed solution

R1 and R2 share exactly one tuple, (b1,5), and the output contains exactly that tuple. This is the set intersection R1∩R2. Algebraically R1−(R1−R2) would also equal the intersection, but option 3 is printed as that equivalent expression; the direct requested operation is option 1, making the item non-unique if option 3 is interpreted literally.

## Why the other options do not work

R1−R2 returns the three R1-only tuples, and the union returns all distinct tuples. Option 3 is mathematically equivalent to intersection, so the source unfortunately provides a second equivalent expression rather than a genuinely distinct distractor.

## Key idea to remember

Intersection keeps tuples present in both relations; A−(A−B)=A∩B.
