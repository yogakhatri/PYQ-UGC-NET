# Question 59

Machine Level Language is a/an

- **1.** Assembly Language
- **2.** Low Level language
- **3.** High level Language
- **4.** Translating Language

## Answer

**2. Low Level language**

## Detailed solution

Machine language consists of binary-encoded instructions executed directly by a processor. It is hardware-dependent and offers almost no abstraction from the instruction-set architecture, so it is classified as a low-level language.

## Why the other options do not work

Assembly language is another low-level language but uses mnemonic symbols and requires an assembler, so it is not identical to machine language. A high-level language provides greater abstraction, and ‘translating language’ is not the relevant language category.

## Key idea to remember

Machine code and assembly are low-level; machine code is the binary form the CPU executes directly.
