# Question 113

Which AI System mimics the evolutionary process to generate increasingly better solutions to a process to a problem? A. Self organizing neural network. B. Back propagation neural network. C. Genetic algorithm. D. Forward propagation neural network. Choose the correct answer from the options given below:

- **1.** A Only
- **2.** B Only
- **3.** C Only
- **4.** D Only

## Answer

**3. C Only**

## Detailed solution

A genetic algorithm evolves a population through selection, crossover, and mutation, retaining fitter candidates across generations. It directly mimics biological evolution.

## Why the other options do not work

The neural-network alternatives learn weights or representations but do not use population-level evolutionary operators.

## Key idea to remember

Evolutionary optimization = population + fitness + selection + variation.
