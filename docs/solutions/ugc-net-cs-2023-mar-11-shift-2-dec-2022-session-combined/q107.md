# Question 107

I. No.5 BID: 18705 Given an expression (A+B*D)/(E-F)+G A. The prefix notation is +/+A*DB-EFG B. The prefix expression is the reverse of the postfix expression C. The order oi operands in intix expression and postiix expression are the same. D. The order of operands in intix expression, pretix expression and postix expression are the same. Choose the most appropriate answer from the options given below:

- **1.** A, C and D Only
- **2.** C and D Only
- **3.** A and B Only
- **4.** A and D Only

## Answer

**1. A, C and D Only**

## Detailed solution

For ((A+B·D)/(E−F))+G, prefix is + / + A * B D − E F G, so A is true. Conversion moves operators but preserves operand order A,B,D,E,F,G, making C and D true. Prefix is not simply reversed postfix, so B is false.

## Why the other options do not work

Options 2 and 4 omit a true statement; option 3 includes the false reversal claim.

## Key idea to remember

Prefix is expression-tree preorder and postfix is postorder; operand order remains stable.
