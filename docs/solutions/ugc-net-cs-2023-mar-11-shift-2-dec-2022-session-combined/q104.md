# Question 104

SB. D:187054 Software project manager is responsible for the following tasks: A. Project Planning B. Project status tracking C. Resource management D. Risk management E. Project delivery within time & budget. Choose the correct answer from the options given below:

- **1.** All the statements are correct.
- **2.** Only B & C are correct.
- **3.** Only A d D are correct
- **4.** All the statements are not correct.

## Answer

**1. All the statements are correct.**

## Detailed solution

A project manager plans work, tracks status, allocates resources, manages risks, and remains accountable for delivery within schedule and budget. All five listed duties belong to the role.

## Why the other options do not work

The restricted subsets omit genuine planning, risk, or delivery responsibilities.

## Key idea to remember

Project management integrates scope, schedule, cost, resources, monitoring, and risk.
