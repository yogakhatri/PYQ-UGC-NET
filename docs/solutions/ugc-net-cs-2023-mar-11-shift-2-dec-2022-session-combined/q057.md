# Question 57

option 1D-32 8870:187007 Simplify the following using K-Map F(A, B,C,D) = Z(0,2,5,7,8,10,13,15)

- **1.** BD + B'D'
- **2.** AC + A'C'
- **3.** BC + B'C'
- **4.** AD + AD

## Answer

**1. BD + B'D'**

## Detailed solution

The minterms split into two groups of four. Minterms 0,2,8,10 all have B=0 and D=0 while A and C vary, giving B′D′. Minterms 5,7,13,15 all have B=1 and D=1 while A and C vary, giving BD. Combining the two implicants gives F=B′D′+BD.

## Why the other options do not work

The minterm membership does not depend on A or C, so expressions built from A,C cannot be correct. The AD expression also changes when A changes, whereas paired minterms show A is eliminated.

## Key idea to remember

In a K-map group, every variable that changes within the group disappears from the implicant.
