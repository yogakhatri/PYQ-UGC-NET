# Question 108

Which of the following is (are) correct about the regular expression? aa*bb*cc*ad* A. The language for the given expression is: B. The Context Free Language for the given expression is: S → AB C A → aAb ab B → cBd cd C→ aCd aDd D → bDc bc C. The language generated by this expression is equal number of 'a's, followed by equal number of 'b's, followed by equal number of 'c's and followed by equal number of'd's. Choose the correct answer from the options given below:

- **1.** Only A is correct
- **2.** Only B is correct
- **3.** Both A and B are correct
- **4.** All the three A, B and C are correct

## Answer

**3. Both A and B are correct**

## Detailed solution

The displayed language description A is the intended set denoted by the expression, and grammar B generates its two component forms through S→AB|C. Statement C imposes one equality constraint across all four symbol counts, which is a different language. Thus A and B, but not C, are correct.

## Why the other options do not work

Options 1 and 2 omit one correct representation; option 4 incorrectly accepts C's all-counts-equal characterization.

## Key idea to remember

Derive the exponent pattern of each grammar branch, then compare it with the stated language.
