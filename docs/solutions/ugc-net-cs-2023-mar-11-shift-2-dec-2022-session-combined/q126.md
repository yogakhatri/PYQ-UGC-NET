# Question 126

SB D:187076 Consider the statements: A. There does not exist a polynomial time algorithm to solve integer linear programming problem. Main focus of PERT is 'minimizing time'. Choose the correct option about the statements A and B.

- **1.** Ais True: B is True
- **2.** A is True: B is False
- **3.** A is False; B is True
- **4.** A is False; B is False

## Answer

**1. Ais True: B is True**

## Detailed solution

General integer linear programming is NP-hard and no polynomial-time algorithm is known (under the standard P≠NP assumption), so A is treated as true in this context. PERT emphasizes schedule/time analysis under uncertain activity durations, so B is also true.

## Why the other options do not work

Options 2–4 reject one or both intended textbook statements.

## Key idea to remember

ILP is computationally hard; PERT is primarily time/schedule oriented.
