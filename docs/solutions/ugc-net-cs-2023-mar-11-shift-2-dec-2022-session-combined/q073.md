# Question 73

I. No.23 )BID:18702: Modifying the software to match changes in the ever changing environment is called as

- **1.** Adaptive maintenance
- **2.** Corrective maintenance
- **3.** Perfective maintenance
- **4.** Preventive maintenance

## Answer

**1. Adaptive maintenance**

## Detailed solution

Adaptive maintenance changes software so it continues to work in a changed environment—for example, after an operating-system, hardware, regulation, or external-interface change. That is exactly the situation described.

## Why the other options do not work

Corrective maintenance fixes discovered faults. Perfective maintenance improves performance or usability or adds enhancements. Preventive maintenance restructures software to reduce the chance of future problems.

## Key idea to remember

Adaptive = environment changed; corrective = defect found; perfective = enhancement; preventive = future-risk reduction.
