# Question 114

Consider the following Learning algorithms. A. Logistic regression. B. Back propagation. C. Linear regression. D. Forward propagation. Which of the following options represents classification algorithm ? Choose the correct answer from the options given below:

- **1.** A& C only
- **2.** B & D only
- **3.** A&B only
- **4.** C & D only

## Answer

**3. A&B only**

## Detailed solution

Logistic regression directly models class probability. Backpropagation trains neural networks used for classification. Linear regression predicts continuous values, while forward propagation is an inference computation rather than a separate learning algorithm. Thus A and B apply.

## Why the other options do not work

Options containing C treat continuous regression as classification; those containing D misclassify an execution phase as a learning algorithm.

## Key idea to remember

Backpropagation learns parameters; forward propagation computes outputs.
