# Question 125

Match List I with List II LIST I LIST II I. Natural language understanding. A. Text planning II. Natural language generation. B. Sentence planning C. Sentence generation D. Map the imput to useful representations Choose the correct answer from the options given below:

- **1.** A-1, B-II, C-I, D-II
- **2.** A-II, B-II, C-I, D-II
- **3.** A-I, B-II, C-II, D-I
- **4.** A-II, B-II, C-II, D-I

## Answer

**4. A-II, B-II, C-II, D-I**

## Detailed solution

Text planning, sentence planning, and sentence generation are successive natural-language-generation activities, so A,B,C→II. Mapping input text into useful semantic representations is natural-language understanding, so D→I.

## Why the other options do not work

The other choices wrongly classify one or more generation stages as understanding.

## Key idea to remember

NLU maps language to meaning; NLG plans content and realizes it as sentences.
