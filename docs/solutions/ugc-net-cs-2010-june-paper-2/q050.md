# Question 50

Data Mining uses ________, ________ and ________ to build effective predictive model. (i) Data set (ii) Information set (iii) Input set (iv) Process set (v) Output set (vi) Test set

- **A.** (i), (ii) and (iv)
- **B.** (ii), (iv) and (v)
- **C.** (i), (v) and (vi)
- **D.** (ii), (iii) and (v)

## Answer

**C. (i), (v) and (vi)**

## Detailed solution

For a predictive model, the data set supplies training examples, the output set supplies the target values/classes the model must predict, and a separate test set measures generalization on unseen examples. Of the offered combinations, i,v,vi is the coherent supervised-learning workflow.

## Why the other options do not work

An 'information set' and 'process set' are not the standard three datasets required to train and evaluate a predictive model. Options A, B, and D omit either the actual data or the held-out test set. The source terminology is loose; modern practice would more precisely say training inputs, target outputs, and validation/test data.

## Key idea to remember

Predictive learning needs examples, targets, and unseen evaluation data.
