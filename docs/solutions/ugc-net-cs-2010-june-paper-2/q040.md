# Question 40

Remote Computing Service involves the use of time sharing and _______.

- **A.** multi-processing
- **B.** interactive processing
- **C.** batch processing
- **D.** real-time processing

## Answer

**B. interactive processing**

## Detailed solution

A remote computing service lets a user interact with a distant computer while its resources are shared among users by time slicing. It therefore combines time sharing with interactive processing.

## Why the other options do not work

Batch processing lacks the interactive remote dialogue; multiprocessing is an implementation possibility rather than the defining service mode; real-time processing imposes deadline constraints not implied here.

## Key idea to remember

Remote time-sharing provides an interactive terminal session on a shared host.
