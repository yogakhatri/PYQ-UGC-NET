# Question 15

The data type created by the data abstraction process is called

- **A.** class
- **B.** structure
- **C.** abstract data type
- **D.** user defined data type

## Answer

**C. abstract data type**

## Detailed solution

Data abstraction specifies a collection of values together with permitted operations while hiding representation details. The resulting conceptual type is called an abstract data type (ADT).

## Why the other options do not work

A class or structure may implement an ADT, and a user-defined type is broader; neither is the precise result named by the abstraction process.

## Key idea to remember

An ADT defines behavior independently of representation.
