# Question 16

An entity instance is a single occurrence of an _______.

- **A.** entity type
- **B.** relationship type
- **C.** entity and relationship type
- **D.** None of these

## Answer

**A. entity type**

## Detailed solution

An entity type describes a category such as Student; an entity instance is one particular occurrence, such as the student with roll number 42. Therefore it is a single occurrence of an entity type.

## Why the other options do not work

A relationship instance is an occurrence of a relationship type, not an entity instance, so B and C mix distinct ER concepts.

## Key idea to remember

Type is the schema-level category; instance is one concrete member.
