# Question 43

Which model is simplest model in Software Development ?

- **A.** Waterfall model
- **B.** Prototyping
- **C.** Iterative
- **D.** None of these

## Answer

**A. Waterfall model**

## Detailed solution

The classic waterfall model is the simplest conceptual lifecycle model: requirements, design, implementation, testing, deployment, and maintenance proceed as a mostly linear sequence of phases.

## Why the other options do not work

Prototyping and iterative models add feedback cycles, repeated refinement, and version management, making their process structure more complex.

## Key idea to remember

Waterfall is the basic linear phase-by-phase development model.
