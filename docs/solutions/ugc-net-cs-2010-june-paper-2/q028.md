# Question 28

End-to-End connectivity is provided from Last-to-Last in

- **A.** Network layer
- **B.** Session layer
- **C.** Transport layer
- **D.** Data link layer

## Answer

**C. Transport layer**

## Detailed solution

The transport layer provides logical end-to-end, process-to-process communication between applications on the two final hosts, including segmentation, reliability/flow control when supported, and port addressing.

## Why the other options do not work

The network layer provides host-to-host packet routing; data link is hop-to-hop; session manages dialogs above transport.

## Key idea to remember

Transport is the OSI end-to-end process-delivery layer.
