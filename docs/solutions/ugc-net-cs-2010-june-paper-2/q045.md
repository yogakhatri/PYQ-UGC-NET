# Question 45

Applications-software

- **A.** is used to control the operating system
- **B.** includes programs designed to help programmers
- **C.** performs a specific task for computer users
- **D.** all of the above

## Answer

**C. performs a specific task for computer users**

## Detailed solution

Application software is created to perform a user-oriented task, such as word processing, accounting, browsing, or image editing. Option C gives this defining property.

## Why the other options do not work

Controlling the operating system is a system-software role. Compilers, debuggers, and similar programmer aids are programming/system tools, so A and B do not define application software and D cannot be true.

## Key idea to remember

Application software serves an end-user task; system software operates/supports the computer platform.
