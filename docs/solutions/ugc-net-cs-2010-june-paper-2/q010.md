# Question 10

The function represented by the k- map given below is BC A 1 0 0 1 1 0 0 1

- **A.** A ⋅ B
- **B.** AB + BC + CA
- **C.** B ⊕ C
- **D.** A ⋅ B ⋅ C

## Answer

**No listed option under standard K-map ordering — the function is C̅**

## Detailed solution

A three-variable K-map orders the BC columns as 00,01,11,10. Both A rows contain 1,0,0,1, so A is irrelevant. The first and fourth columns are adjacent by wraparound and are precisely the columns with C=0; grouping all four 1-cells therefore gives F=C̅. None of A–D states C̅. Option C would follow only if the four columns were incorrectly read in ordinary binary order 00,01,10,11 rather than K-map Gray order.

## Why the other options do not work

A·B, AB+BC+CA, and A·B·C depend on A. (B⊕C)̅ has the Gray-order pattern 1,0,1,0 rather than the shown 1,0,0,1.

## Key idea to remember

Karnaugh-map columns use Gray order so adjacent cells differ in exactly one bit.
