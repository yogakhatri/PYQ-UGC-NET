# Question 41

Software engineering primarily aims on

- **A.** reliable software
- **B.** cost effective software
- **C.** reliable and cost effective software
- **D.** none of the above

## Answer

**C. reliable and cost effective software**

## Detailed solution

Software engineering applies disciplined processes and technical methods so that software both satisfies reliability/quality requirements and can be produced and maintained within realistic cost constraints. The two goals must be balanced, so reliable and cost-effective software is the best answer.

## Why the other options do not work

Reliability without affordability may make a project infeasible, while cheap but unreliable software fails its purpose. Each single-goal option is incomplete.

## Key idea to remember

Engineering optimizes required quality under economic and schedule constraints.
