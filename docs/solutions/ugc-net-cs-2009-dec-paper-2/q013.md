# Question 13

A __________ is a special method used to initialize the instance v ariable of a class.

- **A.** Member function
- **B.** Destructor
- **C.** Constructor
- **D.** Structure

## Answer

**C. Constructor**

## Detailed solution

A constructor is the special class operation invoked during object creation to establish the object's initial state, including initialization of instance variables.

## Why the other options do not work

A generic member function need not initialize an object; a destructor handles cleanup at the end of lifetime; a structure is a type construct, not an initialization method.

## Key idea to remember

Construction initializes an object; destruction releases its resources.
