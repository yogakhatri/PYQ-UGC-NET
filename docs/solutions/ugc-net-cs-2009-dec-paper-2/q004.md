# Question 4

P : “Program is a step by step execution of the instructions”. Given P, which of the following is true ?

- **A.** Program is a subset of an instruction set.
- **B.** Program is a sequence of a subset of an instruction s et.
- **C.** Program is a partially ordered set of an instructi on set.
- **D.** All of the above

## Answer

**D. All of the above**

## Detailed solution

A program draws its operations from the machine/language instruction set, so its distinct instructions form a subset. Execution arranges instruction occurrences in a sequence. Any total sequence order is also a partial order (it satisfies the partial-order conditions with additional comparability). Thus A, B, and C are compatible descriptions and all are true.

## Why the other options do not work

Each of A–C captures only one mathematical aspect—membership, sequence, or ordering—whereas D includes them together.

## Key idea to remember

A sequence is a specially constrained ordered collection drawn from an instruction set.
