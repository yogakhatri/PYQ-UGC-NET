# Question 42

Any error whose cause cannot be identified anywhere within the software system is called ________

- **A.** Internal error
- **B.** External error
- **C.** Inherent error
- **D.** Logic error

## Answer

**B. External error**

## Detailed solution

If investigation finds no cause inside the software system, the failure is attributed to an event or condition outside the program's controlled boundary—an external error. Examples include unavailable devices, malformed external services, or environmental failures.

## Why the other options do not work

Internal and logic errors originate in the program's code/state. An inherent error is rooted in the problem statement or requirements, which is still an identifiable source rather than an unidentified internal cause.

## Key idea to remember

Classify an error by whether its cause lies inside or outside the software boundary.
