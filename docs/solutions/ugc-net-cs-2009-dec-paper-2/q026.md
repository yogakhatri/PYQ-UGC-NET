# Question 26

Device on one network can communicate with devices on anothe r network via a

- **A.** Hub/Switch
- **B.** Utility server
- **C.** File server
- **D.** Gateway

## Answer

**D. Gateway**

## Detailed solution

A gateway connects dissimilar or separately administered networks and forwards/translates traffic between them, allowing a device on one network to communicate with another network.

## Why the other options do not work

A hub or ordinary switch connects devices within a local network segment. Utility and file servers provide services but are not the general inter-network forwarding device.

## Key idea to remember

Inter-network communication requires a layer-3 gateway/router function.
