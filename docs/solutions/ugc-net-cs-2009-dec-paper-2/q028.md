# Question 28

Which of the following are Data Link Layer standards? 1. Ethernet; 2. HSSI; 3. Frame Relay; 4. 10-Base-T; 5. Token Ring.

- **A.** 1, 2, 3
- **B.** 1, 3, 5
- **C.** 1, 3, 4, 5
- **D.** 1, 2, 3, 4, 5

## Answer

**B. 1, 3, 5**

## Detailed solution

Ethernet, Frame Relay, and Token Ring define data-link framing/MAC behavior. HSSI is a high-speed serial physical interface, and 10BASE-T primarily specifies an Ethernet physical-layer medium/signaling implementation. Thus the intended data-link-only set is 1,3,5.

## Why the other options do not work

Options containing item 2 or 4 mix physical-layer interface/media standards into the data-link set.

## Key idea to remember

Separate framing and media-access protocols from physical connectors, signaling, and cabling variants.
