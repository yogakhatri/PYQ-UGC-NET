# Question 110

Which of the following statements) are correct? A. Syntax Directed Translation can be applied in Type Checking. B. Syntax Directed Translation can be used in 3-address code generation. C. Syntax Directed Translation can be used in construction of syntax tree. D. Syntax Directed Translation can be used to recognize valid tokens. Choose the most appropriate answer from the options given below:

- **1.** A, B and C
- **2.** A, B, C and D
- **3.** A, C and D
- **4.** B, C and D

## Answer

**1. A, B and C**

## Detailed solution

Syntax-directed translation attaches semantic actions/attributes to grammar productions and supports type checking, syntax-tree construction, and intermediate three-address-code generation. Token recognition is the lexical analyzer's job, normally specified by regular expressions/finite automata, not SDT. Thus A,B,C.

## Why the other options do not work

Options containing D confuse lexical analysis with syntax-directed semantic processing.

## Key idea to remember

Lexer recognizes tokens; parser plus semantic actions performs syntax-directed translation.
