# Question 96

l. No.46 BID:87041 Inheritance provides the benefits of A. shared interfaces B. run-time polymorphism C. less redundant information D. dynamic binding Choose the Correct answer from the options given below:

- **1.** A. B, C only
- **2.** A, B, D only
- **3.** B, C, D only
- **4.** A, C, D only

## Answer

**1. A. B, C only**

## Detailed solution

Inheritance directly supplies a shared interface and inherited implementation, reducing redundant declarations/implementation (A,C), and supports runtime polymorphism through substitutable derived objects (B). Dynamic binding is the dispatch mechanism used to realize runtime polymorphism rather than an independent inheritance benefit in this classification. Hence A,B,C.

## Why the other options do not work

Options 2–4 omit A, B, or C while treating mechanism D as a separate benefit.

## Key idea to remember

Inheritance promotes reuse and subtype polymorphism; dynamic binding implements virtual dispatch.
