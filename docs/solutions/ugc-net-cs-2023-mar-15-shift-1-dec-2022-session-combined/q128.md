# Question 128

SE 078 Arrange the following search and sort algorithms in ascending order of their worst-case complexities. A. Merge sort B. Linear Search C. Bubble Sort D. Binary Search Choose the correct answer from the options given below:

- **1.** A, B, D, C
- **2.** B, D, A, C
- **3.** B, D, C, A
- **4.** B, A, D, C

## Answer

**No listed option — correct ascending order is D, B, A, C**

## Detailed solution

Worst cases are binary search D=Θ(log n), linear search B=Θ(n), merge sort A=Θ(n log n), bubble sort C=Θ(n²). Ascending order is D,B,A,C, but no option states it.

## Why the other options do not work

Every choice places linear search before binary search or otherwise misorders the growth rates.

## Key idea to remember

log n < n < n log n < n².
