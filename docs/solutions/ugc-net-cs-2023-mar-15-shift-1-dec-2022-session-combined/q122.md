- **2.** A-, B-IV, C-II, D-I|I
- **3.** A-IV, B-I, C-II, D-III
- **4.** A-II, B-III, C-IV, D-I

## Answer

**3. A-IV, B-I, C-II, D-III**

## Detailed solution

8-Queens uses backtracking (A–IV); closest pair uses divide-and-conquer (B–I); string edit distance uses dynamic programming (C–II); fractional knapsack uses a greedy method (D–III).

## Why the other options do not work

Other options assign the standard problems to mismatched paradigms.

## Key idea to remember

Recognize canonical examples for each algorithm-design paradigm.
