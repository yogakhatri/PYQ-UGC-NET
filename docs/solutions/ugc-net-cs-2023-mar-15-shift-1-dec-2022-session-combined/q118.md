# Question 118

SID: D:87068 Match List I with List II List I List I I. Liquid Crystal display A. 2-D Array Input B. Transmissive device II. Flat-bed scanner III. Camera C. Binary image D. 1-D array input IV. Ink jet printer Choose the correct answer from the options given below:

- **1.** A-I, B-III, C-II, D-IV
- **2.** A-IlI, B-I, C-IV, D-I
- **3.** А-III, B-I, C-II, D-IV
- **4.** A-I, B-II, C-IV, D-III

## Answer

**3. А-III, B-I, C-II, D-IV**

## Detailed solution

A camera supplies a 2-D array input (A–III). LCD is transmissive (B–I). A flat-bed scanner produces a binary/digital image (C–II). Ink-jet printing consumes a 1-D stream/array input as listed (D–IV).

## Why the other options do not work

Other mappings interchange capture, display, scanning, and printing devices.

## Key idea to remember

Camera/scanner are image inputs; LCD/inkjet are display/output technologies.
