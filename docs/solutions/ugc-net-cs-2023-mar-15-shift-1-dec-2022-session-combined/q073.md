# Question 73

SB10:87023 Which of the following is a technique used for component wrapping?

- **1.** Black-box wrapping
- **2.** Clear-box wrapping
- **3.** Gray-box wrapping
- **4.** White-box wrapping

## Answer

**1. Black-box wrapping**

## Detailed solution

Black-box wrapping places an adapter around a reusable/legacy component using only its externally visible interface, without modifying its internal design. This is the standard component-wrapping technique intended here.

## Why the other options do not work

The alternative color terms are not the named wrapping technique used in the source's component-reuse taxonomy.

## Key idea to remember

A wrapper adapts an existing component's interface while treating its implementation as a black box.
