# Question 5

*UGC NET CS · 2013 Dec Paper 3 · Estimation and Scheduling · Function-Point Adjustment*

Function points can be calculated by

- **A.** UFP ∗ CAF
- **B.** UFP ∗ FAC
- **C.** UFP ∗ Cost
- **D.** UFP ∗ Productivity

> [!TIP]
> **Correct answer: A. UFP ∗ CAF**

## Solution

Unadjusted Function Points count weighted external inputs, outputs, inquiries, internal logical files and external interface files. The adjusted function-point value is FP=UFP×CAF, where CAF is the complexity adjustment factor derived from general system characteristics.

## Key Points

- Adjusted function points = UFP × complexity adjustment factor (CAF).

## Why the other options are incorrect

FAC reverses the accepted acronym. Cost and productivity may later be estimated using function points, but they are not multipliers in the function-point calculation itself.
