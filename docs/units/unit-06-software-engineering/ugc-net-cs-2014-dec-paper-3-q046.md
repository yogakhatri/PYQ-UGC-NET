# Question 46

*UGC NET CS · 2014 Dec Paper 3 · Software Quality · Reliability as a Probability*

What is the maximum possible value of reliability?

- **A.** 100
- **B.** 10
- **C.** 1
- **D.** 0

> [!TIP]
> **Correct answer: C. 1**

## Solution

Reliability is normally modeled as the probability that a system performs its required function without failure for a specified time under specified conditions. Any probability lies from 0 to 1 inclusive. Perfect reliability is therefore 1, the maximum possible value.

## Key Points

- Reliability is a probability: 0 means no successful operation and 1 means perfect success over the stated mission.

## Why the other options are incorrect

100 would be correct only if reliability were explicitly written as a percentage, but the options use the standard probability-scale value. 10 is outside the probability range, and 0 represents certain failure rather than maximum reliability.
