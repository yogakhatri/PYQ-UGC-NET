# Question 13

*UGC NET CS · 2021 Nov With Answers · Formal Methods · Mathematical Specification Notation*

In software engineering, what kind of notation do formal methods predominantly use?

- **1.** textual
- **2.** diagrammatic
- **3.** mathematical
- **4.** computer code

> [!TIP]
> **Correct answer: 3. mathematical**

## Solution

Formal methods specify and reason about software using mathematically defined syntax and semantics—commonly logic, set theory, relations, algebra, and proof rules. The predominant notation is therefore mathematical, option 3.

## Key Points

- Formal methods replace informal ambiguity with mathematically precise specification and verification.

## Why the other options are incorrect

Text and diagrams may explain a formal model, and code may implement it, but neither provides the precise proof-oriented foundation that makes the method formal.
