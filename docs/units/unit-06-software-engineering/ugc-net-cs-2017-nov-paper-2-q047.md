# Question 47

*UGC NET CS · 2017 Nov Paper 2 · Information Systems · ERP Modules and Enterprise Integration*

Which of the following statements about ERP system is true ?

- **1.** Most ERP software implementations fully achieve seamless integration.
- **2.** ERP software packages are themselves combinations of separate applications for manufacturing, materials/resource planning, general ledger, human resources, procurement and order entry.
- **3.** Integration of ERP systems can be achieved in only one way.
- **4.** An ERP package implemented uniformly throughout an enterprise is likely to contain very flexible connections to allow changes and software variations.

> [!TIP]
> **Correct answer: 2. ERP software packages are themselves combinations of separate applications for manufacturing, materials/resource planning, general ledger, human resources, procurement and order entry.**

## Solution

An enterprise resource planning package integrates a set of functional modules—such as manufacturing, materials planning, finance/general ledger, human resources, procurement, and order management—around shared processes and data. This is precisely the description in option 2.

## Key Points

- ERP unifies multiple business functions through coordinated modules and shared enterprise data, but implementation and integration remain organizational and technical challenges.

## Why the other options are incorrect

ERP implementations do not automatically or universally achieve perfect seamless integration, so option 1 is too absolute. Integration can be implemented in multiple ways, rejecting option 3. Uniform enterprise packages often impose standardized structures and can be difficult to customize; option 4's claim of very flexible connections is not generally true.
