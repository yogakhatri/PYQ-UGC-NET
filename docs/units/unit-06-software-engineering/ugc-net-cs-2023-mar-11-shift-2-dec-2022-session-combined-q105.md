# Question 105

*UGC NET CS · 2023 Mar 11 Shift 2 Dec 2022 Session · Software Configuration Management · Open-source software licensing*

Mr: X designed open source software which must comply with some criteria. Choose the false statement in respect of above. A. No restriction on redistribution of the software as part or whole. B. The integrity of the author's source code must be maintained C. The software can be sold after distribution. Choose the correct answer from the options given below:

- **1.** A only
- **2.** B only
- **3.** C only
- **4.** All statements are false

> [!TIP]
> **Correct answer: No listed option — A, B, and C are all compatible with the Open Source Definition**

## Solution

A matches free redistribution. B reflects the permitted integrity-of-author-source-code condition, under which modifications may be distributed as patches or under changed names/versions. C is also true: open-source software may be sold. None of A–C is false, but no ‘none’ option is supplied.

## Key Points

- Open source permits use, modification, redistribution, and sale; it does not mean zero price.

## Why the other options are incorrect

Options 1–3 each label a valid open-source condition false; option 4 incorrectly says all three are false.
