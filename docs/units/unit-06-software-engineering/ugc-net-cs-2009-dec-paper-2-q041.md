# Question 41

*UGC NET CS · 2009 Dec Paper 2 · Software Process Models · Adaptive Software Development*

Software Engineering is a discipline that integrates _________ for th e development of computer software.

- **A.** Process
- **B.** Methods
- **C.** Tools
- **D.** All

> [!TIP]
> **Correct answer: D. All**

## Solution

Software engineering combines a defined development process, technical methods for analysis/design/testing, and supporting tools for automation and control. These three layers work together to produce and maintain software systematically, so all are integrated.

## Key Points

- Software engineering is commonly described as a layered technology of process, methods, and tools.

## Why the other options are incorrect

Process alone gives management structure, methods alone give techniques, and tools alone automate tasks; none by itself constitutes the full discipline.
