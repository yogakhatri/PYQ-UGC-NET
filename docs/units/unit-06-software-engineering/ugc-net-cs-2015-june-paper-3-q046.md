# Question 46

*UGC NET CS · 2015 June Paper 3 · Software Maintenance · Adaptive, Corrective, and Perfective Maintenance*

Adaptive maintenance is a maintenance which __________.

- **1.** correct errors that were not discovered till testing phase.
- **2.** is carried out to port the existing software to a new environment.
- **3.** improves the system performance.
- **4.** both (2) and (3)

> [!TIP]
> **Correct answer: 2. is carried out to port the existing software to a new environment.**

## Solution

Adaptive maintenance changes software so it continues to operate when its environment changes—for example, porting it to a new operating system, hardware platform, database, or external interface. Therefore option 2 gives the defining purpose.

## Key Points

- Corrective fixes faults; adaptive follows environmental change; perfective improves qualities; preventive reduces future risk.

## Why the other options are incorrect

Fixing latent faults is corrective maintenance. Improving speed, usability, or maintainability is perfective maintenance. Because option 3 is perfective rather than adaptive, the combined option 4 is also wrong.
