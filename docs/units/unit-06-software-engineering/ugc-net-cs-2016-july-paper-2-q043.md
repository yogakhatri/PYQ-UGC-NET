# Question 43

*UGC NET CS · 2016 July Paper 2 · Software Process Models · Waterfall Model versus Risk Analysis*

Which of the following is not included in waterfall model ?

- **1.** Requirement analysis
- **2.** Risk analysis
- **3.** Design
- **4.** Coding

> [!TIP]
> **Correct answer: 2. Risk analysis**

## Solution

The classical waterfall sequence includes requirements analysis, system/software design, implementation (coding), testing, deployment, and maintenance. A separate risk-analysis phase is characteristic of the spiral model, not the waterfall model. Thus option 2 is correct.

## Key Points

- Waterfall is phase-sequential; explicit risk analysis drives the spiral model.

## Why the other options are incorrect

Requirement analysis, design, and coding are core waterfall stages. Choosing any of them would remove an essential step from the sequential development flow.
