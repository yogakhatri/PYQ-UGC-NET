# Question 108

*UGC NET CS · 2018 Dec Paper 1 And 2 · Software Maintenance · Reengineering and Reverse Engineering*

Which of the following statements is/are true? P: Software Reengineering is preferable for software products having high failure rates, having poor design and/or having poor code structure. Q: Software Reverse Engineering is the process of analyzing software with the objective of recovering its design and requirements specification. Choose the correct answer from the code given below :

- **1.** P only
- **2.** Q only
- **3.** Both P and Q
- **4.** Neither P nor Q

> [!TIP]
> **Correct answer: 3. Both P and Q**

## Solution

P is true: frequent failures together with poor design or code structure are strong reasons to reengineer a maintainable but deteriorated system, improving its internal form while preserving needed behaviour. Q is also true: reverse engineering analyzes an existing implementation to recover higher-level information such as architecture, design, data models, and requirements. Thus both P and Q are true, option 3.

## Key Points

- Reverse engineering recovers understanding from code; reengineering uses such understanding to restructure or improve the software.

## Why the other options are incorrect

Options 1 and 2 reject one valid statement, while option 4 rejects both. Reverse engineering is analysis and knowledge recovery; reengineering is the broader examination and alteration used to improve an existing system.
