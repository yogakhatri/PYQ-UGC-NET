# Question 108

*UGC NET CS · 2019 June Paper 1 And 2 · Software Configuration Management · Software maintenance types*

Software products need adaptive maintenance for which of the following reasons?

- **1.** To rectify bugs observed while the system is in use
- **2.** When the customers need the product to run on new platforms
- **3.** To support the new features that users want it to support
- **4.** To overcome wear and tear caused by the repeated use of the software

> [!TIP]
> **Correct answer: 2. When the customers need the product to run on new platforms**

## Solution

Adaptive maintenance modifies software so it continues to work in a changed environment, such as a new operating system, hardware platform, database or external interface. A customer requirement to run on new platforms is therefore adaptive maintenance.

## Key Points

- Corrective fixes defects, adaptive handles environmental change, and perfective improves features or performance.

## Why the other options are incorrect

- **Option 1:** bug repair is corrective maintenance.
- **Option 3:** adding desired features is perfective maintenance.
- **Option 4:** software does not physically wear out through repeated execution.
