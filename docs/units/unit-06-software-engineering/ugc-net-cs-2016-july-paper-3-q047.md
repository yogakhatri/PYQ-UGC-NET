# Question 47

*UGC NET CS · 2016 July Paper 3 · Software Quality and Process Improvement · CMM Maturity Levels*

Which of the following sets represent five stages defined by Capability Maturity Model (CMM) in increasing order of maturity ?

- **1.** Initial, Defined, Repeatable, Managed, Optimized.
- **2.** Initial, Repeatable, Defined, Managed, Optimized.
- **3.** Initial, Defined, Managed, Repeatable, Optimized.
- **4.** Initial, Repeatable, Managed, Defined, Optimized.

> [!TIP]
> **Correct answer: 2. Initial, Repeatable, Defined, Managed, Optimized.**

## Solution

The original five-level Capability Maturity Model progresses as follows: Initial (work is ad hoc), Repeatable (basic project-management practices allow earlier successes to be repeated), Defined (an organization-wide documented process exists), Managed (the process and product are measured and quantitatively controlled), and Optimized (continuous process improvement is institutionalized). This is the order in option 2.

## Key Points

- Remember the CMM ladder as I-R-D-M-O: Initial, Repeatable, Defined, Managed, Optimized.

## Why the other options are incorrect

The Defined level cannot precede Repeatable in this model, because stable project-level discipline is established before an organization-wide standard process. Managed cannot precede Defined, because quantitative control requires a defined process to measure.
