# Question 48

*UGC NET CS · 2012 Dec Paper 2 · Software Quality Assurance · Umbrella Activities*

_____ is an umbrella activity applied throughout the software-engineering process.

- **1.** Debugging
- **2.** Testing
- **3.** Designing
- **4.** Software quality assurance

> [!TIP]
> **Correct answer: 4. Software quality assurance**

## Solution

Software quality assurance (SQA) is called an umbrella activity because it spans the entire software process. It establishes standards, reviews processes and work products, audits compliance, tracks defects and supports continuous quality improvement across communication, planning, modeling, construction and deployment.

## Key Points

- An umbrella activity cuts across every framework phase; SQA provides process-wide quality oversight.

## Why the other options are incorrect

Debugging is performed when faults are located and corrected. Testing executes or evaluates software to reveal failures, and designing creates the solution structure. Each is important but occurs as a specific engineering activity; SQA oversees quality throughout all of them.
