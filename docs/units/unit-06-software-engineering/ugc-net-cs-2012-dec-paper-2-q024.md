# Question 24

*UGC NET CS · 2012 Dec Paper 2 · Software Process Models · Rapid Application Development*

RAD stands for:

- **1.** Rapid and Design
- **2.** Rapid Aided Development
- **3.** Rapid Application Development
- **4.** Rapid Application Design

> [!TIP]
> **Correct answer: 3. Rapid Application Development**

## Solution

RAD stands for Rapid Application Development. It is an iterative, time-boxed software-development approach that emphasizes quick prototyping, strong user feedback, reusable components and frequent delivery instead of a long single-pass lifecycle.

## Key Points

- RAD = Rapid Application Development; connect it with prototypes, user involvement and short development cycles.

## Why the other options are incorrect

'Rapid and Design', 'Rapid Aided Development' and 'Rapid Application Design' are not the expansion of the established software-engineering acronym.
