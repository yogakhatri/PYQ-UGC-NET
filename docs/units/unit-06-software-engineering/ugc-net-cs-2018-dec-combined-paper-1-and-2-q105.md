# Question 105

*UGC NET CS · 2018 Dec Paper 1 And 2 · Software Process Models · Cleanroom Software Engineering*

Which of the following statements is/are false? P: The clean-room strategy to software engineering is based on the incremental software process model. Q: The clean-room strategy to software engineering is one of the ways to overcome "unconscious" copying of copyrighted code. Choose the correct answer from the code given below :

- **1.** P only
- **2.** Q only
- **3.** Both P and Q
- **4.** Neither P nor Q

> [!TIP]
> **Correct answer: 4. Neither P nor Q**

## Solution

P is true: Cleanroom software engineering develops and verifies the system through controlled increments, with formal specification and correctness verification before statistical usage testing. Q is also true: a clean-room approach can separate a team that derives an independent specification from a team that implements from that specification, reducing the risk of consciously or unconsciously copying protected source code. Because the question asks which statements are false and neither is false, option 4 is correct.

## Key Points

- Watch the polarity: both statements are true, so the answer to ‘which are false?’ is ‘neither.’

## Why the other options are incorrect

Option 1 incorrectly marks the incremental-process statement false. Option 2 incorrectly denies the intellectual-property use of clean-room development. Option 3 marks both false even though both describe recognized aspects or applications of the clean-room strategy.
