# Question 45

*UGC NET CS · 2017 Nov Paper 2 · Software Testing · Regression Testing after Maintenance Changes*

Which of the following testing techniques ensures that the software product runs correctly after the changes during maintenance ?

- **1.** Path Testing
- **2.** Integration Testing
- **3.** Unit Testing
- **4.** Regression Testing

> [!TIP]
> **Correct answer: 4. Regression Testing**

## Solution

Regression testing reruns relevant existing tests after a correction, enhancement, refactor, or environmental change. Its purpose is to detect unintended side effects and confirm that previously working behavior still works. Therefore option 4 is correct.

## Key Points

- After any change, regression tests guard against old behavior breaking while new behavior is added or repaired.

## Why the other options are incorrect

Path testing targets control-flow paths, unit testing focuses on individual components, and integration testing focuses on component interfaces. Each can contribute tests to a regression suite, but regression testing is the maintenance-oriented technique named by the question.
