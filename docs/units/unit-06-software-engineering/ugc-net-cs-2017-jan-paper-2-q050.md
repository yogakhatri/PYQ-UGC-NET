# Question 50

*UGC NET CS · 2017 Jan Paper 2 · Information Systems · ERP Integration, Real-Time Access, and Cost*

Which of the following statements is/are true w.r.t. Enterprise Resource Planning (ERP) ? (A) ERP automates and integrates majority of business processes. (B) ERP provides access to information in a Real Time Environment. (C) ERP is inexpensive to implement.

- **1.** (A), (B) and (C) are false.
- **2.** (A) and (B) false; (C) true.
- **3.** (A) and (B) true; (C) false.
- **4.** (A) true; (B) and (C) are false.

> [!TIP]
> **Correct answer: 3. (A) and (B) true; (C) false.**

## Solution

ERP systems integrate data and workflows across functions such as finance, procurement, production, inventory, sales, and human resources, so A is true. A shared integrated database makes current transaction information available across modules in near real time, so B is true in the textbook ERP model. ERP adoption requires software, infrastructure, configuration, data migration, process redesign, training, and change management, so it is not generally inexpensive; C is false. Therefore option 3 is correct.

## Key Points

- ERP's value is enterprise-wide process and data integration; its major tradeoff is costly, complex implementation and organizational change.

## Why the other options are incorrect

Options 1 and 2 reject the core integration and real-time-information benefits of ERP. Option 4 accepts integration but wrongly denies the shared current-information benefit.
