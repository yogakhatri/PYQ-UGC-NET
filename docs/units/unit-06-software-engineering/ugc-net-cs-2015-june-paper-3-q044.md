# Question 44

*UGC NET CS · 2015 June Paper 3 · Software Metrics · Conciseness as Program Compactness*

Which design metric measures the compactness of a program in terms of lines of code?

- **1.** Consistency
- **2.** Conciseness
- **3.** Efficiency
- **4.** Accuracy

> [!TIP]
> **Correct answer: 2. Conciseness**

## Solution

Conciseness measures how compactly a program expresses its required functionality, commonly relating functionality to the number of source lines. Fewer lines for the same delivered behavior indicate greater conciseness, so option 2 matches the question.

## Key Points

- Conciseness is compact expression of functionality; do not confuse it with runtime efficiency.

## Why the other options are incorrect

Consistency concerns uniform conventions, efficiency concerns resource use such as time or memory, and accuracy concerns correctness or error. None directly denotes compactness measured through lines of code.
