# Question 5

*UGC NET CS · 2013 June Paper 3 · Software Design · Golden Rules of Interface Design*

The following three golden rules : (i) Place the user in control (ii) Reduce the user’s memory load (iii) Make the interface consistent are for

- **A.** User satisfaction
- **B.** Good interface design
- **C.** Saving system’s resources
- **D.** None of these

> [!TIP]
> **Correct answer: B. Good interface design**

## Solution

The three rules are a compact framework for user-interface design: give users meaningful control and escape routes, minimize what they must remember between screens, and make commands/layout/feedback consistent. Together they guide the construction of a usable, predictable interface.

## Key Points

- Interface golden rules: user control, low memory burden and consistency.

## Why the other options are incorrect

User satisfaction is an outcome of good design rather than the name of the rules themselves. They do not primarily conserve CPU, memory or storage resources. Because B states their design purpose directly, 'none of these' is false.
