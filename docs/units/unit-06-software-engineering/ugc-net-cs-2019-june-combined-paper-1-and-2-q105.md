# Question 105

*UGC NET CS · 2019 June Paper 1 And 2 · Software Configuration Management · Distributed version control*

Which of the following terms best describes Git?

- **1.** Issue Tracking System
- **2.** Integrated Development Environment
- **3.** Distributed Version Control System
- **4.** Web-based Repository Hosting Service

> [!TIP]
> **Correct answer: 3. Distributed Version Control System**

## Solution

Git is a distributed version-control system. Every normal clone contains repository history and supports local commits, branches and merges without requiring continuous access to a central server.

## Key Points

- Git is the version-control engine; GitHub, GitLab and similar products are hosting and collaboration platforms built around it.

## Why the other options are incorrect

Git itself is not an issue tracker, IDE or hosting service. Platforms such as GitHub and GitLab provide repository hosting and may add issue tracking around Git repositories.
