# Question 47

*UGC NET CS · 2017 Jan Paper 3 · Software Testing · Validation, Verification, Testing, and Debugging*

Complete each of the following sentences in List – I on the left hand side by filling in the word or phrase from the List – II on the right hand side that best completes the sentence : List – I List – II I. Determining whether you have A. Software testing built the right system is called ________ II. Determining whether you have B. Software verification built the system right is called ________ III. ________ is the process of C. Software debugging demonstrating the existence of defects or providing confidence that they do not appear to be present. IV. _______ is the process of D. Software validation discovering the cause of a defect and fixing it. Codes : I II III IV

- **1.** B D A C
- **2.** B D C A
- **3.** D B C A
- **4.** D B A C

> [!TIP]
> **Correct answer: 4. D B A C**

## Solution

Validation asks, 'Did we build the right system?', so I→D. Verification asks, 'Did we build the system right?', so II→B. Software testing executes or evaluates the software to reveal defects or build confidence that failures are absent, so III→A. Debugging locates a defect's cause and repairs it, so IV→C. The sequence D,B,A,C is option 4.

## Key Points

- Validation=right product; verification=product built right; testing=find failures; debugging=find and fix causes.

## Why the other options are incorrect

The other codes reverse validation and verification or confuse defect discovery with defect repair. Testing can expose failure evidence; debugging begins after a defect is suspected and traces and fixes its cause.
