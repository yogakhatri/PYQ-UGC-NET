# Question 43

*UGC NET CS · 2014 Dec Paper 2 · Software Design · Software Architecture Definition*

The software _________ of a program or a computing system is the struc ture or structures of the system, which comprise software components, the externall y visible properties of those components, and the relationships among them.

- **A.** Design
- **B.** Architecture
- **C.** Process
- **D.** Requirement

> [!TIP]
> **Correct answer: B. Architecture**

## Solution

Software architecture is the structure or set of structures of a system, including its software components, their externally visible properties and the relationships among them. 'Externally visible' means properties other components may rely on, such as services, interfaces, behavior and constraints.

## Key Points

- Architecture captures components, externally visible properties and relationships at the system-structure level.

## Why the other options are incorrect

Design is broader and includes internal detailed decisions beyond the definition quoted. Process describes how software is developed. Requirements state needed behavior and qualities, not the implemented structural organization.
