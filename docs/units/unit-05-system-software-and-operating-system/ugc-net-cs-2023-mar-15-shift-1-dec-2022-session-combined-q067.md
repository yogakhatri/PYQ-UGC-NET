# Question 67

*UGC NET CS · 2023 Mar 15 Shift 1 Dec 2022 Session · Security · Access-matrix protection model*

stoption 10-457611 51 0:87017 Model of protection can be viewed as

- **1.** Owner matrix
- **2.** Domain matrix
- **3.** Access Matrix
- **4.** Global matrix

> [!TIP]
> **Correct answer: 3. Access Matrix**

## Solution

The protection model is represented by an access matrix whose rows are protection domains, columns are objects, and cells contain permitted operations.

## Key Points

- Access matrix unifies ACL (columns) and capability-list (rows) views.

## Why the other options are incorrect

Owner/domain/global matrix are not the standard general model.
