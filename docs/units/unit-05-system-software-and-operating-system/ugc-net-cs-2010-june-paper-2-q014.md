# Question 14

*UGC NET CS · 2010 June Paper 2 · Threads · Multicore Programming*

Match the following. A. Garbage collection in; B. Nameless object; C. Template support; D. A forward reference; E. Derived class inherits from base class. 1. Java; 2. generic programming; 3. defines a class; 4. member function; 5. within a statement.

- **A.** 1,5,4,2,3
- **B.** 1,5,2,3,4
- **C.** 5,1,2,3,4
- **D.** 5,4,3,1,2

> [!TIP]
> **Correct answer: B. 1,5,2,3,4**

## Solution

Garbage collection is characteristic of Java (A-1). A temporary/nameless object can be created within a statement (B-5). Templates support generic programming (C-2). A forward reference appears while defining a class/member before its later declaration (D-3 as intended). A derived class inherits base-class member functions (E-4). The sequence is 1,5,2,3,4.

## Key Points

- Java→GC, template→generic programming, derivation→inherited members.

## Why the other options are incorrect

The other mappings confuse templates with member functions, garbage collection with statement temporaries, or inheritance with class-definition terminology.
