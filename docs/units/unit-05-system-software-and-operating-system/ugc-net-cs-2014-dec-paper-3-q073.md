# Question 73

*UGC NET CS · 2014 Dec Paper 3 · Windows Operating Systems · Built-In Volume Shrink and Extend*

Which listed Windows version contains a built-in partition manager that can shrink and extend existing volumes?

- **A.** Windows Vista
- **B.** Windows 2000
- **C.** Windows NT
- **D.** Windows 98

> [!TIP]
> **Correct answer: A. Windows Vista**

## Solution

Windows Vista's built-in Disk Management added the ability to shrink and extend supported existing volumes through the graphical operating-system tool. This allowed repartitioning available space without requiring the older versions listed or a separate third-party partition manager. Thus Vista is the intended answer.

## Key Points

- Historical Windows cue: Vista Disk Management introduced built-in shrink/extend operations for existing supported volumes.

## Why the other options are incorrect

Windows 2000's Disk Management had more limited volume-management capabilities and did not provide Vista's general shrink-volume operation. Windows NT and Windows 98 likewise lacked the listed built-in shrink-and-extend facility. Only A matches the feature combination.
