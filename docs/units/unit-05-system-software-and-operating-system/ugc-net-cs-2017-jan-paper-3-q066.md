# Question 66

*UGC NET CS · 2017 Jan Paper 3 · Operating-System Concepts · Single-User and Multiuser Historical Systems*

Names of some of the Operating Systems are given below : (a) MS-DOS (b) XENIX (c) OS/2 In the above list, following operating systems didn’t provide multiuser facility.

- **1.** (a) only
- **2.** (a) and (b) only
- **3.** (b) and (c) only
- **4.** (a), (b) and (c)

> [!TIP]
> **Correct answer: No valid option: MS-DOS and OS/2 lacked general multiuser operation; XENIX was multiuser, so the correct set is (a) and (c).**

## Solution

MS-DOS was designed as a single-user system. XENIX was Microsoft's Unix-derived multiuser, multitasking system. IBM's OS/2 was a multitasking but single-user operating system rather than a general simultaneous multiuser system. Therefore the systems that did not provide multiuser capability are MS-DOS and OS/2—(a) and (c)—but that combination is missing from the choices.

## Key Points

- Multitasking means one user can run multiple tasks; multiuser means multiple users have separately managed simultaneous sessions and resources.

## Why the other options are incorrect

Option 1 omits OS/2. Options 2, 3, and 4 wrongly include XENIX. The question likely confused multitasking, which OS/2 supported, with multiuser operation, which is a different property.
