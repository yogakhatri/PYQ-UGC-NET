# Question 74

*UGC NET CS · 2014 Dec Paper 3 · System Security · Trojan, Virus and Worm Distinctions*

A Trojan horse is

- **A.** A program that appears to perform a legitimate function but also has a hidden component used for malicious purposes such as attacking message security or impersonation.
- **B.** A piece of code that can attach itself to other programs in the system and spread to other systems when programs are copied or transferred.
- **C.** A program that spreads to other computer systems by exploiting security holes, such as weaknesses in remote-process facilities.
- **D.** All of the above

> [!TIP]
> **Correct answer: A. A program that appears to perform a legitimate function but also has a hidden component used for malicious purposes such as attacking message security or impersonation.**

## Solution

A Trojan horse disguises malicious behavior inside software that appears legitimate or useful. It relies on the user or system accepting and executing the program; its hidden payload may steal data, impersonate a user, or create unauthorized access. Option A gives this definition.

## Key Points

- Trojan disguises; virus attaches; worm self-propagates over networks.

## Why the other options are incorrect

B describes a virus, which attaches to host programs and spreads with them. C describes a worm, which self-propagates across systems by exploiting network or remote-execution weaknesses. Since those are distinct malware classes, all of the above is false.
