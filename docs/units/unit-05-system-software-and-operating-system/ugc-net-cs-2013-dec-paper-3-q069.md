# Question 69

*UGC NET CS · 2013 Dec Paper 3 · Windows Operating Systems · Object Linking and Embedding*

To place a sound into a word document, following feature of windows is used :

- **A.** Clipboard
- **B.** Task switching
- **C.** C Win App
- **D.** OLE

> [!TIP]
> **Correct answer: D. OLE**

## Solution

Object Linking and Embedding (OLE) lets one Windows application place an object created or managed by another application inside its document. An audio object can therefore be embedded in a Word document and activated through its associated server/application.

## Key Points

- OLE enables compound documents containing linked or embedded objects such as audio, spreadsheets or drawings.

## Why the other options are incorrect

The Clipboard transfers copied data but does not define a persistent linked or embedded compound-document object. Task switching merely changes the active application. 'C Win App' is not the Windows compound-document feature described.
