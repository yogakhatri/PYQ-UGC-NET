# Question 89

*UGC NET CS · 2023 Mar 15 Shift 1 Dec 2022 Session · Digital Logic Circuits and Components · Logic Gates*

No-39039 Fuzzy logic is a form of

- **1.** Two valued logic
- **2.** Crisp set logic
- **3.** Binary set logic
- **4.** Many valued logic

> [!TIP]
> **Correct answer: 4. Many valued logic**

## Solution

Fuzzy logic permits truth degrees across an interval such as [0,1], rather than restricting every proposition to exactly true or false. It is therefore many-valued logic.

## Key Points

- Fuzzy truth is graded, not merely 0 or 1.

## Why the other options are incorrect

Two-valued, crisp, and binary logics allow only two membership/truth states.
