# Question 90

*UGC NET CS · 2023 Mar 15 Shift 1 Dec 2022 Session · Data Representation · Data Types*

510-82040 The 4Ws problem canvas helps in identifying the key elements related to the problem. 4Ws problem canvas is a part of

- **1.** Problem scoping
- **2.** Data acquisition
- **3.** Data modeling
- **4.** Problem Evaluation

> [!TIP]
> **Correct answer: 1. Problem scoping**

## Solution

The 4Ws canvas—Who, What, Where, Why—is used at the start of an AI/problem-solving project to define stakeholders, context, and the problem's boundaries. It belongs to problem scoping.

## Key Points

- 4Ws clarifies the problem before data or model work begins.

## Why the other options are incorrect

Data acquisition, modeling, and evaluation occur after the problem has been scoped.
