# Question 38

*UGC NET CS · 2012 Dec Paper 2 · Digital Logic Circuits and Components · TTL, ECL and CMOS Logic Families*

Match IC families with their basic circuits: a. TTL; b. ECL; c. CMOS. 1. NAND; 2. NOR; 3. Inverter.

- **1.** a-1,b-2,c-3
- **2.** a-3,b-2,c-1
- **3.** a-2,b-3,c-1
- **4.** a-2,b-1,c-3

> [!TIP]
> **Correct answer: 1. a-1,b-2,c-3**

## Solution

The representative basic gate for TTL is the NAND gate, whose multi-emitter input structure naturally supports TTL logic. ECL is commonly built around the NOR/OR function using differential transistor stages. The fundamental CMOS building block is the inverter, made from complementary pMOS and nMOS transistors. Thus a-1, b-2, c-3.

## Key Points

- Logic-family anchors: TTL-NAND, ECL-NOR, CMOS-inverter.

## Why the other options are incorrect

Each other code assigns at least one family to the wrong characteristic circuit. In particular, the CMOS inverter is the clearest anchor; once c-3 is fixed, only option 1 also gives the standard TTL-NAND and ECL-NOR associations.
