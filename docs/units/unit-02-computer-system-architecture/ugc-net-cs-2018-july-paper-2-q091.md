# Question 91

*UGC NET CS · 2018 July Paper 2 · Memory Hierarchy · Battery-Backed CMOS RAM*

In the conventional PC-motherboard context, CMOS configuration memory is a type of:

- **1.** RAM
- **2.** ROM
- **3.** EPROM
- **4.** Auxiliary storage

> [!TIP]
> **Correct answer: 1. RAM**

## Solution

In traditional PCs, `CMOS` commonly refers to a small amount of battery-backed CMOS RAM that stores firmware setup data such as clock and hardware configuration. It is writable and loses its contents without sustained power, so among the options it is RAM, option 1.

## Key Points

- PC `CMOS memory` is battery-backed volatile RAM used for firmware configuration and the real-time clock.

## Why the other options are incorrect

ROM and EPROM are nonvolatile memory types and are not what the conventional phrase `CMOS setup memory` denotes. Auxiliary storage means devices such as disks. Technically CMOS is a semiconductor fabrication technology, but the motherboard usage in this question specifically means CMOS RAM.
