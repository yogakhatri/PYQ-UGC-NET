# Question 12

*UGC NET CS · 2012 Dec Paper 2 · Central Processing Unit · Indexed Addressing*

In which addressing mode is the effective address of the operand generated by adding a constant value to the contents of a register?

- **1.** Absolute
- **2.** Indirect
- **3.** Immediate
- **4.** Index

> [!TIP]
> **Correct answer: 4. Index**

## Solution

The described calculation is EA = contents of register + constant displacement. This is indexed addressing (also called displacement or base-plus-offset addressing in many architectures): the instruction supplies the constant and a register supplies the variable/base part of the address.

## Key Points

- Recognize addressing modes from the EA formula: register + constant offset indicates indexed/displacement addressing.

## Why the other options are incorrect

Absolute addressing uses an address stated directly in the instruction. Indirect addressing obtains the effective address through another memory/register reference rather than the stated register-plus-constant calculation. Immediate addressing places the operand value itself in the instruction, so no effective memory address is formed.
