# Question 1

*UGC NET CS · 2016 July Paper 3 · Digital Logic Circuits · Combinational versus Sequential Circuits*

Which of the following is a sequential circuit ?

- **1.** Multiplexer
- **2.** Decoder
- **3.** Counter
- **4.** Full adder

> [!TIP]
> **Correct answer: 3. Counter**

## Solution

A sequential circuit has memory: its output or next state depends on the current input and the stored previous state. A counter is built from flip-flops, and its next binary count depends on its present count. Therefore the counter is sequential.

## Key Points

- Combinational output depends only on present inputs; sequential behavior also depends on stored state.

## Why the other options are incorrect

A multiplexer selects one current input, a decoder maps a current input code to output lines, and a full adder computes sum and carry from its current inputs. In their standard forms these are combinational circuits with no stored state.
