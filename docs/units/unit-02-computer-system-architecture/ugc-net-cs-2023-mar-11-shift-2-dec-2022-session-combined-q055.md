# Question 55

*UGC NET CS · 2023 Mar 11 Shift 2 Dec 2022 Session · Memory Hierarchy · Cache Memory*

Which one of the following types of memory is fastest

- **1.** Cache Memory
- **2.** Register Memory
- **3.** Main Memory
- **4.** Secondary Memory

> [!TIP]
> **Correct answer: 2. Register Memory**

## Solution

CPU registers are located inside the processor and are accessed directly by instructions. They have the smallest capacity but the lowest access latency in the usual memory hierarchy. Cache is also fast, but it sits below registers; main memory and secondary storage are progressively slower.

## Key Points

- Typical speed order: registers → cache → main memory → secondary storage.

## Why the other options are incorrect

Cache memory is designed to reduce average memory-access time but is not faster than the processor’s registers. Main memory requires an external memory access, while secondary memory is slower still.
