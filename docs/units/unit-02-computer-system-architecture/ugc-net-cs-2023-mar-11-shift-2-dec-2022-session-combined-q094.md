# Question 94

*UGC NET CS · 2023 Mar 11 Shift 2 Dec 2022 Session · Memory Hierarchy · DRAM and SRAM comparison*

Which of the following is/are true for Dynamic RAM (DRAM)? A. It is slower than static RAM(SRAM) B. Packing Density is higher than Static RAM (SRAM) C. It is faster than static RAM(SRAM) D. It requires data retreshing Choose the correct answer from the options given below:

- **1.** A, B and D Only
- **2.** A. Band C Only
- **3.** C and D Only
- **4.** B and D Only

> [!TIP]
> **Correct answer: 1. A, B and D Only**

## Solution

DRAM stores each bit as charge in a compact capacitor cell, giving greater density than SRAM, but it is slower and the charge must be periodically refreshed. Hence A, B, and D are true while C is false.

## Key Points

- DRAM: dense, inexpensive, slower, refresh required; SRAM: faster, larger cell, no refresh.

## Why the other options are incorrect

Every other choice includes C or omits A even though SRAM's latch cells are faster than refreshed capacitor cells.
