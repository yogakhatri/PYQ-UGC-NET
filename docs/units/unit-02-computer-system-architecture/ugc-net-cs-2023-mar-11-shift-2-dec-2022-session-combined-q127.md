# Question 127

*UGC NET CS · 2023 Mar 11 Shift 2 Dec 2022 Session · Basic Computer Organization and Design · Fetch-decode-execute cycle*

The correct sequence in fetch-execute cycle is A. Decode B. Fetch C. Execute Chose the correct answer from the following

- **1.** A-B-C
- **2.** B-C-A
- **3.** C-B-A
- **4.** B-A-C

> [!TIP]
> **Correct answer: 4. B-A-C**

## Solution

The processor first fetches the instruction, decodes its opcode and operands, and then executes it. With B=Fetch, A=Decode, C=Execute, the sequence is B–A–C.

## Key Points

- Instruction cycle: fetch → decode → execute.

## Why the other options are incorrect

Every other sequence decodes or executes an instruction before it has been fetched.
