# Question 47

*UGC NET CS · 2012 June Paper 2 · Basic Computer Organization and Design · Arithmetic and Logic Unit*

Where does a computer add and compare data ?

- **A.** Hard disk
- **B.** Floppy disk
- **C.** CPU chip
- **D.** Memory chip

> [!TIP]
> **Correct answer: C. CPU chip**

## Solution

Arithmetic addition and data comparison are carried out by the arithmetic and logic unit inside the CPU chip. The control unit selects the operation and registers supply operands; the ALU produces the arithmetic result or condition flags used by comparisons.

## Key Points

- The CPU's ALU performs arithmetic and logical comparisons.

## Why the other options are incorrect

Hard and floppy disks store data persistently. Memory chips store instructions/data and return them to the CPU; ordinary memory does not perform the requested addition and comparison.
