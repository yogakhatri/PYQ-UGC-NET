# Question 37

*UGC NET CS · 2012 June Paper 2 · Central Processing Unit · Software Interrupts*

Interrupts which are initiated by an instruction are

- **A.** Internal
- **B.** External
- **C.** Hardware
- **D.** Software

> [!TIP]
> **Correct answer: D. Software**

## Solution

A software interrupt is generated by executing a specific instruction such as INT, trap or a system-call instruction. It deliberately transfers control to an interrupt/trap handler using the processor's interrupt mechanism. Therefore an interrupt initiated by an instruction is classified as software.

## Key Points

- Hardware interrupts arrive from outside the instruction stream; software interrupts are explicitly invoked by instructions.

## Why the other options are incorrect

External/hardware interrupts originate from devices, timers or external pins. 'Internal' is often used for exceptions caused by execution conditions, such as divide-by-zero, but the explicit interrupt-instruction case is conventionally called a software interrupt.
