# Question 3

*UGC NET CS · 2012 June Paper 2 · Microprogrammed Control · Conditional Branching*

The branch logic that provides decision-making capabilities in the control unit is known as:

- **A.** Controlled transfer
- **B.** Conditional transfer
- **C.** Unconditional transfer
- **D.** None of the above

> [!TIP]
> **Correct answer: B. Conditional transfer**

## Solution

Decision-making requires the next control address to depend on a tested status bit or condition. A conditional transfer selects either the sequential next address or a branch address according to that condition. This is the branch logic that lets a control unit choose between alternative microinstruction sequences.

## Key Points

- Conditional branching tests a status condition and chooses the next microinstruction address.

## Why the other options are incorrect

An unconditional transfer always jumps and therefore makes no true/false decision. 'Controlled transfer' is not the standard name for the conditional branch operation described. Hence 'none' is also false.
