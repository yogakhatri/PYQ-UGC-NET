# Question 46

*UGC NET CS · 2009 Dec Paper 2 · Pipeline and Vector Processing · Instruction Pipeline*

The single stage network is also called

- **A.** one sided network
- **B.** two sided network
- **C.** recirculating network
- **D.** pipeline network

> [!TIP]
> **Correct answer: C. recirculating network**

## Solution

A single-stage interconnection network may need to send an item through the same switching stage several times before it reaches its destination. Because data can circulate through that stage repeatedly, it is called a recirculating network.

## Key Points

- Repeated passes through one switching stage give the name recirculating network.

## Why the other options are incorrect

One-sided/two-sided describe neither this routing behavior nor the standard name, and a pipeline normally uses a sequence of distinct stages.
