# Question 93

*UGC NET CS · 2023 Mar 11 Shift 2 Dec 2022 Session · Memory Hierarchy · Virtual-memory purpose and storage*

Which of the statement is not true in regards to virtual memory?

- **1.** The main objective for using virtual memory is to increase the effective capacity of the memory system
- **2.** Size of the program can be as large as the size of the secondary memory
- **3.** Program and data are stored in the secondary memory
- **4.** None of these

> [!TIP]
> **Correct answer: 4. None of these**

## Solution

Virtual memory increases the effective addressable memory, keeps nonresident program/data pages in secondary storage, and permits programs larger than physical main memory (subject to virtual-address and backing-store limits). Thus statements 1–3 express the intended properties and none is false.

## Key Points

- Virtual memory separates a process's address space from the amount of installed RAM.

## Why the other options are incorrect

Selecting 1, 2, or 3 would deny a core feature of demand-paged virtual memory.
