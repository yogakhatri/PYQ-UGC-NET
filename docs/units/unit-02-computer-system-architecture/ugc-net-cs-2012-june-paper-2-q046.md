# Question 46

*UGC NET CS · 2012 June Paper 2 · Basic Computer Organization and Design · Microcomputer System Components*

A complete microcomputer system consists of

- **A.** Microprocessor
- **B.** Memory
- **C.** Peripheral equipment
- **D.** All of the above

> [!TIP]
> **Correct answer: D. All of the above**

## Solution

A usable microcomputer system needs a microprocessor to execute instructions, memory to hold programs and data, and peripheral/I/O equipment so it can communicate with users, storage and external devices. The processor alone is only the CPU component, not the complete system.

## Key Points

- Computer system=processor+memory+input/output/peripheral subsystem.

## Why the other options are incorrect

Each of A-C names one necessary component but omits the other two. Therefore only 'all of the above' describes the complete system.
