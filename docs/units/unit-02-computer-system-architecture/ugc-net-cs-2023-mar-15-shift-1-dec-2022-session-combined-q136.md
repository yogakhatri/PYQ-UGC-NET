# Question 136

*UGC NET CS · 2023 Mar 15 Shift 1 Dec 2022 Session · Memory Hierarchy · Virtual Memory*

Given below are the two statements: Statement I: The main objective for using virtual memory is to increase the effective capacity of the memory system Statement II : The size of virtual memory can be larger than the main memory In the light of the above statements, choose the correct answer from the options given below

- **1.** Both Statement I and Statement Il are true.
- **2.** Both Statement I and Statement Il are false.
- **3.** Statement I is true but Statement Il is false.
- **4.** Statement I is false but Statement Il is true.

> [!TIP]
> **Correct answer: 1. Both Statement I and Statement Il are true.**

## Solution

Virtual memory uses secondary storage and address translation to give a process an address space that can exceed the currently installed physical RAM. It therefore increases the effective memory capacity visible to programs (Statement I), and its virtual address space may be larger than main memory (Statement II). Both are true.

## Key Points

- Virtual address space is an abstraction whose size is not limited to the amount of physical RAM currently resident.

## Why the other options are incorrect

Options 2–4 incorrectly deny one or both defining properties of virtual memory.
