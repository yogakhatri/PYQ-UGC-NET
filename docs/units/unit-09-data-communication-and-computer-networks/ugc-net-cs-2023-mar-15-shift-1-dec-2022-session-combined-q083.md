# Question 83

*UGC NET CS · 2023 Mar 15 Shift 1 Dec 2022 Session · OSI and TCP/IP Layer Functions · Address Resolution Protocol*

Which protocol is used to acquire the MAC Address of a host whose IP address is known?

- **1.** Dynamic Host control protocol
- **2.** Address Resolution Protocol
- **3.** Reverse Address Resolution Protocol
- **4.** User Datagram Protocol

> [!TIP]
> **Correct answer: 2. Address Resolution Protocol**

## Solution

ARP broadcasts a request for a known IPv4 address on the local link; the owner replies with its MAC address.

## Key Points

- Known IP → local MAC uses ARP.

## Why the other options are incorrect

RARP performs the historical reverse mapping, DHCP assigns configuration, and UDP is a transport protocol.
