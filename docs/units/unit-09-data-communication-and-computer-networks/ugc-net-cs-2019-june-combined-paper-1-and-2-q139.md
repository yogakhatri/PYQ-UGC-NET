# Question 139

*UGC NET CS · 2019 June Paper 1 And 2 · Network Security · IP Spoofing*

The ability to inject Internet packets carrying a false source IP address is known as:

- **1.** Man-in-the-middle attack
- **2.** IP phishing
- **3.** IP sniffing
- **4.** IP spoofing

> [!TIP]
> **Correct answer: 4. IP spoofing**

## Solution

IP spoofing is the creation or injection of an IP packet whose source-address field has been forged. The receiver then sees a claimed source different from the system that actually transmitted the packet. It may be used to hide origin or support reflection and impersonation attacks.

## Key Points

- Spoofing falsifies identity information; IP spoofing specifically falsifies a source IP address.

## Why the other options are incorrect

A man-in-the-middle attacker intercepts or alters communication between parties. Phishing deceives users into revealing information. Sniffing observes network traffic. None of those terms specifically means forging the packet's source IP field.
