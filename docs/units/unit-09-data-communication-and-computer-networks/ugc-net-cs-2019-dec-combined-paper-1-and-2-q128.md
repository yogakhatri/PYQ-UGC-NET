# Question 128

*UGC NET CS · 2019 Dec Paper 1 And 2 · Cloud Computing and IoT · Cloud Platforms and Content Delivery*

Consider the following statements. (a) Windows Azure is a cloud-based operating system. (b) Google App Engine is an integrated set of online services for consumers to communicate and share with others. (c) Amazon CloudFront is a web service for content delivery. Which statement(s) are correct?

- **1.** Only (a) and (b)
- **2.** Only (a) and (c)
- **3.** Only (b) and (c)
- **4.** (a), (b) and (c)

> [!TIP]
> **Correct answer: 2. Only (a) and (c)**

## Solution

Statements (a) and (c) are correct in the terminology used when these services were introduced. Windows Azure was described as Microsoft's cloud operating system and platform for running and managing applications in Microsoft data centres. Amazon CloudFront is a content-delivery-network service: it caches and serves content through geographically distributed edge locations. Statement (b) describes a consumer communication or social-services suite, not Google App Engine. App Engine is a platform-as-a-service environment in which developers deploy applications while Google manages much of the underlying infrastructure. Hence only (a) and (c) are correct, giving option 2.

## Key Points

- Distinguish cloud roles: Azure was presented as a cloud OS/platform, App Engine is PaaS, and CloudFront is a CDN.

## Why the other options are incorrect

Options 1 and 4 incorrectly treat Google App Engine as a set of consumer sharing services. Option 3 rejects the historically correct description of Windows Azure as a cloud-based operating platform.
