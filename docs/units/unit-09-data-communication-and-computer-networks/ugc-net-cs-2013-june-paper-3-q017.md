# Question 17

*UGC NET CS · 2013 June Paper 3 · Data Communication · Baud Rate and Bit Rate*

The baud rate of a signal is 600 baud/second. If each signal unit carries 6 bits, then the bit rate of a signal is ________.

- **A.** 3600
- **B.** 100
- **C.** 6/600
- **D.** None of the above

> [!TIP]
> **Correct answer: A. 3600**

## Solution

Baud counts signal units (symbols) per second, while bit rate counts bits per second. If each of 600 symbols per second represents 6 bits, bit rate=600×6=3600 bits/s.

## Key Points

- Bit rate = baud rate × bits carried by each signal unit.

## Why the other options are incorrect

100 results from dividing rather than multiplying. 6/600 has the wrong operation and units. Since 3600 is listed, 'none of the above' is false.
