# Question 44

*UGC NET CS · 2018 July Paper 2 · Wireless Networks · Cellular Generations, AMPS and IS-95*

Which statements are true? (a) AMPS is a second-generation cellular system. (b) IS-95 is a second-generation cellular system based on CDMA and DSSS. (c) Third-generation cellular systems aim to provide universal personal communication.

- **1.** (a) and (b) only
- **2.** (b) and (c) only
- **3.** (a), (b) and (c)
- **4.** (a) and (c) only

> [!TIP]
> **Correct answer: 2. (b) and (c) only**

## Solution

Statement (a) is false: AMPS is an analog first-generation (1G) cellular system. Statement (b) is true: IS-95 is a 2G cdmaOne standard using code-division multiple access with direct-sequence spread-spectrum techniques. Statement (c) describes the 3G vision of universal personal/mobile communications and is true in the historical taxonomy. Therefore (b) and (c) only are true, option 2.

## Key Points

- 1G AMPS was analog; 2G IS-95 was digital CDMA; 3G targeted broader universal mobile multimedia service.

## Why the other options are incorrect

Options 1, 3, and 4 all include the false classification of AMPS as 2G. Digital systems such as IS-95 belong to the second generation; AMPS predates them as analog 1G.
