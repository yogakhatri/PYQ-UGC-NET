# Question 108

*UGC NET CS · 2020 Nov With Answers · Cloud Computing · Virtualization and Resource Pooling*

Which cloud concept is related to pooling and sharing resources? (A) Virtual memory, (B) service, (C) virtualization.

- **1.** (C) only
- **2.** (A) and (B) only
- **3.** (A) only
- **4.** (B) only

> [!TIP]
> **Correct answer: 1. (C) only**

## Solution

Virtualization abstracts physical compute, storage, and network resources into logical instances that can be allocated among multiple tenants or workloads. That abstraction is the enabling cloud concept for pooling and sharing a common physical resource base, so C is correct. Virtual memory is an operating-system memory abstraction, and ‘service’ describes how capability is offered to a consumer rather than the mechanism that pools infrastructure. Therefore C only, option 1.

## Key Points

- Virtualization decouples logical resources from physical hardware, allowing the hardware pool to be shared safely and flexibly.

## Why the other options are incorrect

Options 2–4 omit virtualization. Virtual memory can share RAM within a system, but it is not the cloud concept meant by resource pooling; a service may consume pooled resources but does not itself perform the pooling.
