# Question 114

*UGC NET CS · 2018 Dec Paper 1 And 2 · Cloud Computing and IoT · SaaS, PaaS and IaaS Service Models*

A cloud contains a software stack such as operating systems and application software. This is referred to as which model?

- **1.** SaaS
- **2.** PaaS
- **3.** IaaS
- **4.** MaaS

> [!TIP]
> **Correct answer: 1. SaaS**

## Solution

The clue includes not only the operating system and runtime stack but also the application software delivered from the cloud. That corresponds to Software as a Service (SaaS): the consumer uses the provider's running application while the provider manages the underlying platform and infrastructure. Therefore option 1 is the intended answer.

## Key Points

- SaaS delivers the finished application and the stack beneath it; PaaS delivers a development/deployment platform; IaaS delivers virtualized infrastructure.

## Why the other options are incorrect

With PaaS, the provider supplies an application-hosting platform but the consumer deploys its own application. With IaaS, the consumer controls the operating system and deployed applications above virtualized compute, storage, and networking. MaaS is not the applicable one of the three core NIST cloud service models.
