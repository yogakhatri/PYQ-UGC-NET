# Question 18

*UGC NET CS · 2013 June Paper 3 · OSI and TCP/IP Layer Functions · Layer Responsibilities*

Match the following : a. Data link layer i. Flow control b. Network layer ii. Node to node delivery c. Transport layer iii. Mail services d. Application layer iv. Routing Codes : a b c d

- **A.** ii i iv iii
- **B.** ii iv i iii
- **C.** ii i iii iv
- **D.** ii iv iii i

> [!TIP]
> **Correct answer: B. ii iv i iii**

## Solution

The data-link layer provides node-to-node delivery over one link, so a→ii. The network layer chooses routes across networks, so b→iv. The transport layer provides end-to-end flow control, so c→i. Application-layer protocols provide user services such as electronic mail, so d→iii. The sequence is ii, iv, i, iii.

## Key Points

- Link=node-to-node; network=routing; transport=end-to-end flow control; application=mail and user services.

## Why the other options are incorrect

A assigns routing to transport, C assigns mail services to transport/routing to application, and D assigns flow control to application. Only B preserves all four standard layer responsibilities.
