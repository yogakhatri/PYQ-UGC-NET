# Question 27

*UGC NET CS · 2017 Nov Paper 2 · Routing Algorithms · Non-Adaptive and Static Routing*

__________ do not take their decisions on measurements or estimates of the current traffic and topology.

- **1.** Static algorithms
- **2.** Adaptive algorithms
- **3.** Non - adaptive algorithms
- **4.** Recursive algorithms

> [!TIP]
> **Correct answer: No unique answer: options 1 and 3 are synonyms in routing; non-adaptive algorithms are also called static routing algorithms.**

## Solution

The standard classification contrasts adaptive routing, which changes decisions using current topology or traffic information, with non-adaptive routing, whose routes are computed in advance. Non-adaptive routing is commonly called static routing. Therefore the quoted definition directly names option 3, but it also describes option 1 in the networking context, so the choices are not mutually exclusive.

## Key Points

- Adaptive/dynamic routing reacts to network state; non-adaptive/static routing uses routes fixed or computed in advance.

## Why the other options are incorrect

Option 2 is the opposite: adaptive routing responds to changes. Option 4 is unrelated; recursion is a program/algorithm structure, not this routing classification. An intended answer key will normally choose option 3 because it matches the textbook term verbatim.
