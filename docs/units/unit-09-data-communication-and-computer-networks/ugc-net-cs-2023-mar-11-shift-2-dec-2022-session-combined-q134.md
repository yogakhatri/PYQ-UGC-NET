# Question 134

*UGC NET CS · 2023 Mar 11 Shift 2 Dec 2022 Session · Data Communication · Digital and Analog Transmission*

Rearrange the following sequence in the context of OSI Layers: A. Transforming the raw bits in the form of frame for transmission B. Transmission of raw bits over communication channel C. Handling user interfaces D. Control and monitoring of subnet E. Transmission data through connection oriented or connection less using datagrams Choose the correct answer from the options given below:

- **1.** A-B-C-D-E
- **2.** B-C-D-E-A
- **3.** B-A-D-E-C
- **4.** A-B-D-E-C

> [!TIP]
> **Correct answer: 3. B-A-D-E-C**

## Solution

Bottom-up OSI order is physical transmission of bits (B), data-link framing (A), network-layer subnet control (D), transport connection/datagram service (E), and application/user interface (C). Hence B–A–D–E–C.

## Key Points

- Physical → Data Link → Network → Transport → upper/user layers.

## Why the other options are incorrect

Other sequences place framing before physical signaling or user-interface handling below network/transport functions.
