# Question 15

*UGC NET CS · 2015 June Paper 3 · Data Communication · Quantization of Analog Signals*

The process of dividing an analog signal into a string of discrete outputs, each of constant amplitude, is called :

- **1.** Strobing
- **2.** Amplification
- **3.** Conditioning
- **4.** Quantization

> [!TIP]
> **Correct answer: 4. Quantization**

## Solution

Quantization maps each sampled analog amplitude to one of a finite set of allowed amplitude levels. The output is therefore a sequence of discrete-valued, constant level representatives. This is the amplitude-discretization stage used in analog-to-digital conversion.

## Key Points

- Sampling discretizes time; quantization discretizes amplitude.

## Why the other options are incorrect

Strobing or sampling discretizes time, not amplitude. Amplification changes signal magnitude continuously. Conditioning filters or scales a signal to make it suitable for processing but does not by itself map amplitudes to a finite set.
