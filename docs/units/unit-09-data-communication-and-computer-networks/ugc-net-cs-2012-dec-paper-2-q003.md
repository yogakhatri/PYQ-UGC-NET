# Question 3

*UGC NET CS · 2012 Dec Paper 2 · Mobile Technology · GSM Architecture*

The GSM network is divided into which three major systems?

- **A.** SS, BSS, OSS
- **B.** BSS, BSC, MSC
- **C.** CELL, BSC, OSS
- **D.** SS, CELL, MSC

> [!TIP]
> **Correct answer: A. SS, BSS, OSS**

## Solution

The standard high-level GSM architecture has three major subsystems: the Switching System (SS), which performs switching and subscriber/mobility functions; the Base Station System (BSS), which provides the radio access path; and the Operation and Support System (OSS), which supervises and maintains the network. Individual elements such as the MSC and BSC live inside these larger systems.

## Key Points

- Remember GSM at the top level as SS + BSS + OSS: switching, radio access, and operations/support.

## Why the other options are incorrect

B mixes the BSS with two components, BSC and MSC, instead of naming the three major systems. C uses a cell and a BSC, which are radio concepts/components rather than all three subsystems. D likewise mixes systems and a cell while omitting BSS and OSS.
