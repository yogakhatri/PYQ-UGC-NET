# Question 45

*UGC NET CS · 2012 June Paper 2 · Mobile Technology · Global Positioning System*

_____ is a satellite-based tracking system that enables determination of a person's position.

- **1.** Bluetooth
- **2.** WAP
- **3.** Short Message Service
- **4.** Global Positioning System

> [!TIP]
> **Correct answer: 4. Global Positioning System**

## Solution

The Global Positioning System uses signals from a constellation of satellites. A receiver measures signal travel-time differences from multiple satellites and solves for its three-dimensional position and clock offset. It is therefore the satellite-based positioning/tracking system described.

## Key Points

- GPS determines location by ranging to multiple navigation satellites.

## Why the other options are incorrect

Bluetooth is a short-range local wireless technology. WAP is an application protocol framework for mobile web access. SMS carries short text messages over cellular networks. None independently performs satellite positioning.
