# Question 50

*UGC NET CS · 2014 Dec Paper 2 · Mobile Networks · GSM Interfaces, Protocols and CDMA*

Match the following : List – I List – II a. Call control protocol i. Interface between Base Transceiver Stat ion (BTS) and Base Station Controller (BSC) b. A-bis ii. Spread spectrum c. BSMAP iii. Connection management d. CDMA iv. Works between Mobile Switching Centre (MSC) and Base Station Subsystem (BSS) Codes : a b c d

- **A.** iii iv i ii
- **B.** iii i iv ii
- **C.** i ii iii iv
- **D.** iv iii ii i _______________

> [!TIP]
> **Correct answer: B. iii i iv ii**

## Solution

Call-control protocol provides connection management, so a→iii. A-bis is the GSM interface between the Base Transceiver Station and Base Station Controller, so b→i. BSMAP operates between the Mobile Switching Centre and Base Station Subsystem, so c→iv. CDMA is a spread-spectrum access method, so d→ii. The sequence is iii, i, iv, ii.

## Key Points

- GSM cues: A-bis=BTS↔BSC; BSMAP=MSC↔BSS; CDMA=spread spectrum; call control=connection management.

## Why the other options are incorrect

A swaps the A-bis and BSMAP roles. C maps every item to the wrong generic order. D assigns call control to the MSC–BSS interface and CDMA to the BTS–BSC interface, both incorrect.
