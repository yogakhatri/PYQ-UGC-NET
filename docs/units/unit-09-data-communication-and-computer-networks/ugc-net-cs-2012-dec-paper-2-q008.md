# Question 8

*UGC NET CS · 2012 Dec Paper 2 · Mobile Technology · GSM MAP and SS7*

The Mobile Application Part (MAP) typically runs on top of which protocol?

- **1.** SNMP (Simple Network Management Protocol)
- **2.** SMTP (Simple Mail Transfer Protocol)
- **3.** SS7 (Signalling System 7)
- **4.** HTTP (Hypertext Transfer Protocol)

> [!TIP]
> **Correct answer: 3. SS7 (Signalling System 7)**

## Solution

Mobile Application Part (MAP) is a telecommunications application-layer protocol used for GSM/UMTS mobility, authentication, roaming and short-message-related signaling between network databases and switches. It belongs to the Signalling System No. 7 protocol family and is carried through the SS7 transaction/signaling stack, traditionally over TCAP.

## Key Points

- GSM MAP is part of the SS7 signaling environment; associate it with mobility and subscriber-database transactions, not Internet application protocols.

## Why the other options are incorrect

SNMP manages IP-network devices, SMTP transfers electronic mail, and HTTP transfers web resources. None provides the telecom signaling services on which GSM MAP operates.
