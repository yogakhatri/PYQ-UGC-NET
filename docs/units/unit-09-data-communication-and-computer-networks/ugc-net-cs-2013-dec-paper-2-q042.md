# Question 42

*UGC NET CS · 2013 Dec Paper 2 · Mobile Technology · WAP Protocol Stack*

Match the following : List – I List – II a. Wireless Application Environment i. HTTP b. Wireless Transaction Protocol ii. IP c. Wireless Datagram Protocol iii. Scripts d. Wireless iv. UDP Codes : a b c d

- **A.** ii iv i iii
- **B.** iv iii ii i
- **C.** iv iii i ii
- **D.** iii i iv ii

> [!TIP]
> **Correct answer: D. iii i iv ii**

## Solution

WAE contains WML/WMLScript facilities, so it matches Scripts (iii). WTP supplies transaction services analogous to web request handling and is matched with HTTP (i) in this comparison. WDP provides a datagram service analogous to UDP, so c-iv. The remaining printed 'Wireless' item maps to IP (ii). The code is therefore iii, i, iv, ii.

## Key Points

- WAP stack cues: WAE hosts WMLScript; WTP handles transactions; WDP offers UDP-like datagrams over wireless bearers.

## Why the other options are incorrect

The other codes fail at least the two strongest anchors: WAE must map to scripts and WDP to UDP. The source itself truncates list item d to 'Wireless', so the last pairing is obtained by the one-to-one matching after those protocol correspondences are fixed.
