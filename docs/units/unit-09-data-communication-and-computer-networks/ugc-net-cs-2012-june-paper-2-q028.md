# Question 28

*UGC NET CS · 2012 June Paper 2 · Mobile Technology · Mobile IP Agent Discovery and Registration*

Mobile IP provides two basic functions.

- **A.** Route discovery and registration
- **B.** Agent discovery and registration
- **C.** IP binding and registration
- **D.** None of the above

> [!TIP]
> **Correct answer: B. Agent discovery and registration**

## Solution

Mobile IP first performs agent discovery: a mobile node learns whether it is on its home or a foreign network and discovers mobility agents through advertisements/solicitations. It then registers its care-of address with its home agent so packets can be tunnelled to its current location. These are the two basic control functions.

## Key Points

- Mobile IP operation begins with agent discovery and care-of-address registration.

## Why the other options are incorrect

Route discovery belongs to routing protocols, not the named Mobile IP pair. 'IP binding' describes an association but is not paired as the standard basic function here. Hence 'none' is false.
