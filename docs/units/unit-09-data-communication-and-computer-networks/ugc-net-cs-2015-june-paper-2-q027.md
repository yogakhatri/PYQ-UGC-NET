# Question 27

*UGC NET CS · 2015 June Paper 2 · Application-Layer Protocols · Well-Known Port Numbers*

Match the following port numbers with their uses : List - I List - II (a) 23 (i) World wide web (b) 25 (ii) Remote Login (c) 80 (iii) USENET news (d) 119 (iv) E - mail Codes : (a) (b) (c) (d)

- **1.** (iv) (i) (ii) (iii)
- **2.** (ii) (i) (iv) (iii)
- **3.** (ii) (iv) (iii) (i)
- **4.** (ii) (iv) (i) (iii)

> [!TIP]
> **Correct answer: 4. (ii) (iv) (i) (iii)**

## Solution

The standard associations are TCP port 23 for Telnet remote login, TCP port 25 for SMTP e-mail transfer, port 80 for HTTP on the World Wide Web, and TCP port 119 for NNTP/USENET news. Thus `(a),(b),(c),(d)` map to `(ii),(iv),(i),(iii)`, which is option 4.

## Key Points

- Memorize the core ports here as 23–Telnet, 25–SMTP, 80–HTTP, and 119–NNTP.

## Why the other options are incorrect

Each other code sequence swaps at least two well-known services. A useful memory anchor is `23 Telnet, 25 SMTP, 80 HTTP, 119 NNTP`; applying it directly leaves only option 4.
