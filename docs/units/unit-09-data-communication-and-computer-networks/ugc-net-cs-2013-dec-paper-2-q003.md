# Question 3

*UGC NET CS · 2013 Dec Paper 2 · Data Communication · Baud Rate and Bit Rate*

An analog signal carries 4 bits in each signal unit. If 1000 signal units are sent per second, then baud rate and bit rate of the signal are _______ and _______.

- **A.** 4000 bauds \ sec & 1000 bps
- **B.** 2000 bauds \ sec & 1000 bps
- **C.** 1000 bauds \ sec & 500 bps
- **D.** 1000 bauds \ sec & 4000 bps

> [!TIP]
> **Correct answer: D. 1000 bauds \ sec & 4000 bps**

## Solution

Baud rate counts signal units (symbols) transmitted per second, so 1000 signal units per second means 1000 baud. Each signal unit represents 4 bits, making the bit rate 1000 symbols/s x 4 bits/symbol = 4000 bps.

## Key Points

- Bit rate = baud rate x bits per signal unit.
- Here 1000 x 4 = 4000 bps.

## Why the other options are incorrect

The other choices either interchange baud and bit rate or fail to multiply symbols per second by bits per symbol. Baud equals bit rate only when every symbol carries one bit.
