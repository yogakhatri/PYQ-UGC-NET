# Question 48

*UGC NET CS · 2017 Jan Paper 2 · Mobile Computing · Mobile-Workforce and Location-Aware Applications*

Which of the following is/are true w.r.t. applications of mobile computing ? (A) Travelling of salesman (B) Location awareness services

- **1.** (A) true; (B) false.
- **2.** Both (A) and (B) are true.
- **3.** Both (A) and (B) are false.
- **4.** (A) false; (B) true.

> [!TIP]
> **Correct answer: 2. Both (A) and (B) are true.**

## Solution

A travelling salesperson or field-service worker benefits from mobile access to customer records, orders, schedules, inventory, and communication while away from an office, so A is a mobile-computing application. Location-aware services use a device's current position to provide navigation, nearby search, tracking, or contextual content, so B is also an application. Both are true, giving option 2.

## Key Points

- Mobile computing supports users on the move; location awareness is one important context-sensitive capability within it.

## Why the other options are incorrect

Options 1 and 4 incorrectly reject one established use case, while option 3 rejects both. Mobility supports both work performed while travelling and services whose behavior depends on location.
