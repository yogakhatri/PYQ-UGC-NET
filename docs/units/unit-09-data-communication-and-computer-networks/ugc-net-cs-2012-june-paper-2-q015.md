# Question 15

*UGC NET CS · 2012 June Paper 2 · OSI and TCP/IP Layer Functions · Presentation-Layer Functions*

Decryption and encryption of data are the responsibility of which of the following layer ?

- **A.** Physical layer
- **B.** Data Link layer
- **C.** Presentation layer
- **D.** Session layer

> [!TIP]
> **Correct answer: C. Presentation layer**

## Solution

In the OSI model, the Presentation layer handles the representation of application data. Its standard responsibilities include translation/format conversion, compression and encryption/decryption before data is passed between the Application and Session layers.

## Key Points

- OSI Presentation layer: translation, compression and encryption.

## Why the other options are incorrect

The Physical layer transmits signals and bits. The Data Link layer frames local-link data and detects link errors. The Session layer establishes and coordinates dialogues/checkpoints; encryption is conventionally assigned to Presentation.
