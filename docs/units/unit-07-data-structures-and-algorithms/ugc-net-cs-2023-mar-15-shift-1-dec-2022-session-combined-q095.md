# Question 95

*UGC NET CS · 2023 Mar 15 Shift 1 Dec 2022 Session · Data Structures · Arrays and Applications*

BID:87045 Which of the following statement is/are True? A. The processor has direct access to both primary and secondary memory B. Primary memory stores the active instructions and data for the program being executed on the process C. Secondary memory is used as a backup memory D. Memory system is implemented on a single level memory Choose the correct answer from the options given below:

- **1.** A and B only
- **2.** Band Conly
- **3.** A, B and C only
- **4.** A and D Only

> [!TIP]
> **Correct answer: 2. Band Conly**

## Solution

The CPU directly accesses primary memory, not disk/secondary storage, so A is false. Primary memory holds active instructions/data (B), and secondary storage provides persistent/backup storage (C). Memory is hierarchical, not single-level, so D is false. B and C only are true.

## Key Points

- CPU-addressable primary memory and I/O-accessed secondary storage are distinct hierarchy levels.

## Why the other options are incorrect

Other options include false A or D or omit B/C.
