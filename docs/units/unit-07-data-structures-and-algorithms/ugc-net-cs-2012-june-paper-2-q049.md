# Question 49

*UGC NET CS · 2012 June Paper 2 · Data Structures · Linear Data Structures*

Which of the following data structure is linear type ?

- **A.** Strings
- **B.** Lists
- **C.** Queues
- **D.** All of the above

> [!TIP]
> **Correct answer: D. All of the above**

## Solution

A linear data structure arranges elements in a logical sequence so, except at ends, each element has a predecessor and successor. Strings are sequences of characters, lists are sequences of elements, and queues are ordered sequences with insertion/removal rules. All three are linear structures.

## Key Points

- Strings, lists and queues all organize data sequentially.

## Why the other options are incorrect

A, B and C are each correct examples, but selecting only one omits the others. Trees and graphs would be standard nonlinear contrasts.
