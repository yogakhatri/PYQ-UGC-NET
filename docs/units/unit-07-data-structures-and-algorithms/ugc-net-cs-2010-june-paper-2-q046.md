# Question 46

*UGC NET CS · 2010 June Paper 2 · Complexity Theory · NP-Completeness*

The cost of the network is usually determined by

- **A.** time complexity
- **B.** switching complexity
- **C.** circuit complexity
- **D.** none of these

> [!TIP]
> **Correct answer: B. switching complexity**

## Solution

For an interconnection/switching network, hardware cost is commonly modeled by switching complexity—the number and complexity of switching elements and interconnections required to realize its connectivity.

## Key Points

- Network topology cost grows with the switching fabric needed to implement it.

## Why the other options are incorrect

Time complexity measures algorithmic running time. 'Circuit complexity' is broader and less specific than the standard switching-network cost measure used in this context.
