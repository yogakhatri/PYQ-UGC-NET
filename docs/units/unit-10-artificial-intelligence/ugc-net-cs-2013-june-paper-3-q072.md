# Question 72

*UGC NET CS · 2013 June Paper 3 · Knowledge Representation · Frames for Stereotyped Situations*

Which of the following is a knowledge representation technique used to represent knowledge about stereotype situation ?

- **A.** Semantic network
- **B.** Frames
- **C.** Scripts
- **D.** Conceptual Dependency

> [!TIP]
> **Correct answer: B. Frames**

## Solution

A frame is a structured collection of slots, values, defaults and attached procedures used to describe a stereotyped object or situation. A 'restaurant table,' 'classroom' or 'room' frame can supply normal attributes while allowing instance-specific values to override defaults. That matches the wording 'stereotype situation.'

## Key Points

- Frames model stereotyped situations; scripts model stereotyped event sequences.

## Why the other options are incorrect

A semantic network emphasizes concepts and labeled relations. A script specifically represents a stereotyped sequence of events, such as the steps of dining at a restaurant, rather than a situation's slot structure. Conceptual dependency represents sentence meaning through primitive actions.
