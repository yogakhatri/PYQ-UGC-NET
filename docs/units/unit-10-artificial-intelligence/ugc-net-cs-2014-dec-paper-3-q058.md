# Question 58

*UGC NET CS · 2014 Dec Paper 3 · Knowledge Representation · Scripts, Frames, Conceptual Dependencies and Associative Networks*

Match the following : a. Script i. Directed graph with labelled nodes for graphical representation of knowledge b. Conceptual Dependencies ii. Knowledge about objects and events is stored in record-like structures consisting of slots and slot values. c. Frames iii. Primitive concepts and rules to represent natural language statements d. Associative Network iv. Frame like structures used to represent stereotypical patterns for commonly occurring events in terms of actors, roles, props and scenes Codes : a b c d

- **A.** iv ii i iii
- **B.** iv iii ii i
- **C.** ii iii iv i
- **D.** i iii iv ii

> [!TIP]
> **Correct answer: B. iv iii ii i**

## Solution

A script represents a stereotyped event sequence with actors, roles, props, and scenes, so a→iv. Conceptual dependency uses primitive actions and rules to represent the meaning of natural-language statements, b→iii. Frames are record-like slot-and-value structures for objects or events, c→ii. An associative/semantic network is a labeled directed graph, d→i. The code is iv, iii, ii, i.

## Key Points

- Script = stereotyped event; conceptual dependency = primitive semantic acts; frame = slots; associative network = labeled graph.

## Why the other options are incorrect

A confuses conceptual dependencies with frames and associative networks. C assigns scripts the frame definition and frames the script definition. D assigns graph structure to scripts and slot structure to associative networks. B preserves all four standard representations.
