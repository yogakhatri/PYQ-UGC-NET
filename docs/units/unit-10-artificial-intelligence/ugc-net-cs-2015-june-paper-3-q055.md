# Question 55

*UGC NET CS · 2015 June Paper 3 · Knowledge Representation · Data–Information–Knowledge–Intelligence Hierarchy*

Match the following : List - I List - II (a) Intelligence (i) Contextual, tacit, transfer needs learning (b) Knowledge (ii) Scattered facts, easily transferable (c) Information (iii) Judgemental (d) Data (iv) Codifiable, endorsed with relevance and purpose Codes : (a) (b) (c) (d)

- **1.** (iii) (ii) (iv) (i)
- **2.** (iii) (i) (iv) (ii)
- **3.** (i) (ii) (iii) (iv)
- **4.** (i) (iii) (iv) (ii)

> [!TIP]
> **Correct answer: 2. (iii) (i) (iv) (ii)**

## Solution

Intelligence applies judgement, so (a)–(iii). Knowledge is contextual and often tacit; transferring it requires learning, so (b)–(i). Information is data organized and codified with relevance and purpose, so (c)–(iv). Raw data consists of scattered facts that are relatively easy to copy or transfer, so (d)–(ii). The code is `(iii),(i),(iv),(ii)`.

## Key Points

- Data are facts; information gives relevance; knowledge adds context and learning; intelligence applies judgement.

## Why the other options are incorrect

The other sequences confuse raw facts with contextual knowledge or judgment. Following the progression data → information → knowledge → intelligence makes option 2 the only consistent mapping.
