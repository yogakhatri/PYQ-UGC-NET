# Question 71

*UGC NET CS · 2014 Dec Paper 3 · Fuzzy Sets · Crossover Points*

A ______ point of a fuzzy set A is a point x∈X at which μ_A(x)=0.5.

- **A.** core
- **B.** support
- **C.** crossover
- **D.** α-cut

> [!TIP]
> **Correct answer: C. crossover**

## Solution

A crossover point of a fuzzy set is an element whose membership grade is exactly 0.5. It marks the halfway level between full nonmembership and full membership. Therefore x with μ_A(x)=0.5 is a crossover point.

## Key Points

- Fuzzy-set landmarks: support μ>0, core μ=1, crossover μ=0.5, α-cut μ≥α.

## Why the other options are incorrect

The core contains elements with membership 1. The support contains elements with membership greater than 0. An α-cut is a set of elements meeting a threshold μ_A(x)≥α (or >α for a strong cut), not the name of one point specifically at 0.5.
