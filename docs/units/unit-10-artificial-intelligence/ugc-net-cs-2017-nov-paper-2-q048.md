# Question 48

*UGC NET CS · 2017 Nov Paper 2 · Machine Learning · Clustering versus Classification Methods*

Which of the following is not a Clustering method ?

- **1.** K-means method
- **2.** Self-organizing feature map method
- **3.** K-nearest-neighbor method
- **4.** Agglomerative method

> [!TIP]
> **Correct answer: 3. K-nearest-neighbor method**

## Solution

K-nearest neighbors (KNN) is normally a supervised method: it predicts the class or value of a new example from nearby labeled training examples. It is therefore not a clustering method in the standard classification, making option 3 correct.

## Key Points

- Clustering discovers unlabeled groups; KNN uses labeled neighbors to classify or regress a new instance.

## Why the other options are incorrect

K-means partitions observations into clusters around learned centroids. Self-organizing maps provide unsupervised topology-preserving organization and can reveal clusters. Agglomerative clustering builds a hierarchy by repeatedly merging the closest clusters.
