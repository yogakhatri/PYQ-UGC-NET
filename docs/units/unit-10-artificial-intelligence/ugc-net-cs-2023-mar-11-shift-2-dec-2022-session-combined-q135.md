# Question 135

*UGC NET CS · 2023 Mar 11 Shift 2 Dec 2022 Session · Artificial Neural Networks · Unsupervised and Reinforcement Learning*

Choose the correct option describing the features of an artificial neural network. A. It is essentially a machine-learning algorithm. B. It is useful when solving problems for which the data set is very large. C. It can extract features without input from the programmer. D. These are systems modeled on the human brain and nervous system.

- **1.** All the statements are correct
- **2.** Only B and C are correct
- **3.** Only A and D are correct
- **4.** All the statements are not correct

> [!TIP]
> **Correct answer: 1. All the statements are correct**

## Solution

ANNs are machine-learning models inspired by nervous systems, scale to large data, and multilayer networks can learn useful features rather than requiring every feature to be hand-programmed. A–D are all correct at the intended level.

## Key Points

- Representation learning is a central advantage of modern neural networks.

## Why the other options are incorrect

Options 2–4 incorrectly reject genuine ANN characteristics.
