# Question 75

*UGC NET CS · 2016 July Paper 3 · Knowledge Representation · Expert-System Components*

A software program that infers and manipulates existing knowledge in order to generate new knowledge is known as :

- **1.** Data dictionary
- **2.** Reference mechanism
- **3.** Inference engine
- **4.** Control strategy

> [!TIP]
> **Correct answer: 3. Inference engine**

## Solution

An inference engine is the reasoning component of a knowledge-based or expert system. It applies rules or other inference procedures to facts in the knowledge base, derives consequences, and adds newly inferred knowledge. That is precisely the function described, so option 3 is correct.

## Key Points

- Knowledge base stores facts and rules; inference engine applies them; control strategy helps choose which inference to perform next.

## Why the other options are incorrect

A data dictionary describes data definitions and metadata. 'Reference mechanism' is not the reasoning component named here. A control strategy determines the order in which rules or search alternatives are considered, but the inference engine is the program that performs the overall infer-and-manipulate operation.
