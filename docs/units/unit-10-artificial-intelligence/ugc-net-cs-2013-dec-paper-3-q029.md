# Question 29

*UGC NET CS · 2013 Dec Paper 3 · Fuzzy Sets · Normal and Subnormal Fuzzy Sets*

The height h(A) of a fuzzy set A is defined as h(A) = sup A( x) x ∈ A Then the fuzzy set A is called normal when

- **A.** h(A) = 0
- **B.** h(A) < 0
- **C.** h(A) = 1
- **D.** h(A) < 1

> [!TIP]
> **Correct answer: C. h(A) = 1**

## Solution

The height of a fuzzy set is the supremum of its membership grades. A fuzzy set is normal when at least one element has full membership grade 1, so its height equals 1.

## Key Points

- Normal fuzzy set: max/sup membership is 1.
- Subnormal: every membership is below 1.

## Why the other options are incorrect

Membership grades lie between 0 and 1, making a negative height impossible. Height 0 is the empty fuzzy set, while height below 1 describes a subnormal fuzzy set.
