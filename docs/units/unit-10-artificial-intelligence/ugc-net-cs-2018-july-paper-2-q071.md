# Question 71

*UGC NET CS · 2018 July Paper 2 · Intelligent Agents · Observable and Deterministic Environments*

In artificial intelligence, an environment is uncertain if it is ________.

- **1.** Not fully observable and not deterministic
- **2.** Not fully observable or not deterministic
- **3.** Fully observable but not deterministic
- **4.** Not fully observable but deterministic

> [!TIP]
> **Correct answer: 2. Not fully observable or not deterministic**

## Solution

An agent faces uncertainty if it lacks complete state information or cannot predict the result of an action with certainty. The first occurs in a partially observable environment; the second in a nondeterministic environment. Either condition is enough, so `not fully observable or not deterministic` is correct, option 2.

## Key Points

- Uncertainty can come from incomplete perception OR nondeterministic action outcomes.

## Why the other options are incorrect

Option 1 requires both sources of uncertainty simultaneously and is therefore too restrictive. Options 3 and 4 describe valid particular cases, but neither states the general condition covering both kinds.
