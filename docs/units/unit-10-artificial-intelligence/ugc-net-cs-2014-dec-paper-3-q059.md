# Question 59

*UGC NET CS · 2014 Dec Paper 3 · Knowledge Representation · Expert-System Components*

Match the following components of an expert system : a. I/O interface i. Accepts user’s queries and responds to question through I/O interface b. Explanation module ii. Contains facts and rules about the domain c. Inference engine iii. Gives the user, the ability to follow inferencing steps at any time during consultation d. Knowledge base iv. Permits the user to communicate with the system in a natural way Codes : a b c d

- **A.** i iii iv ii
- **B.** iv iii i ii
- **C.** i iii ii iv
- **D.** iv i iii ii

> [!TIP]
> **Correct answer: B. iv iii i ii**

## Solution

The I/O interface lets the user communicate naturally with the system, so a→iv. The explanation module exposes the reasoning steps, b→iii. The inference engine accepts a query and applies reasoning to produce a response, c→i. The knowledge base stores domain facts and rules, d→ii. The sequence is iv, iii, i, ii.

## Key Points

- Expert system flow: interface receives, inference engine reasons over the knowledge base, explanation module shows why/how.

## Why the other options are incorrect

A gives the I/O interface the inference engine's processing role. C assigns knowledge-base content to the inference engine and communication to the knowledge base. D swaps explanation and inference responsibilities. B keeps interface, explanation, reasoning, and stored knowledge distinct.
