# Question 85

*UGC NET CS · 2020 Nov With Answers · Artificial Intelligence · Abstraction in State-Space Representation*

The process of removing details from a given state representation is called

- **1.** Extraction
- **2.** Mining
- **3.** Selection
- **4.** Abstraction

> [!TIP]
> **Correct answer: 4. Abstraction**

## Solution

Abstraction deliberately suppresses details that are irrelevant to the current reasoning task while retaining the features needed to solve it. In state-space problem solving, this produces a simpler representation, often reducing the number of distinctions and transitions the search must consider. Thus the described process is abstraction, option 4.

## Key Points

- Abstraction means retaining essential structure while hiding problem details that do not affect the solution.

## Why the other options are incorrect

Extraction retrieves selected information, mining discovers patterns in data, and selection chooses among existing items. None of these terms specifically denotes constructing a less detailed state representation.
