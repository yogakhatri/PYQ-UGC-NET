# Question 148

*UGC NET CS · 2019 June Paper 1 And 2 · Artificial Neural Networks · Unsupervised Learning and Self-Organizing Maps*

Which of the following is an example of an unsupervised neural network?

- **1.** Back-propagation network
- **2.** Hebb network
- **3.** Associative-memory network
- **4.** Self-organizing feature map

> [!TIP]
> **Correct answer: 4. Self-organizing feature map**

## Solution

A self-organizing feature map, or Kohonen map, learns without labelled target outputs. Neurons compete to represent an input, and the winning neuron and its neighbours adapt, organizing similar inputs into nearby map regions. It is therefore the clearest unsupervised network among the options.

## Key Points

- SOMs use competitive, neighbourhood-preserving learning to cluster unlabelled inputs.

## Why the other options are incorrect

Back-propagation is ordinarily taught as supervised error learning using targets. Hebbian rules and associative memories can appear in different learning settings, but the named network class specifically designed and conventionally classified here as unsupervised is the self-organizing feature map.
