# Question 73

*UGC NET CS · 2015 Dec Paper 3 · Regular Languages · Language Definitions and String Membership*

Let A={0^⌊n/2⌋ 1^n : n≥0} over Σ={0,1}, where ⌊n/2⌋ is the floor of n/2. Which string is not in A?

- **1.** 011
- **2.** 0111
- **3.** 0011
- **4.** 001111

> [!TIP]
> **Correct answer: 3. 0011**

## Solution

A string in A has n ones and exactly ⌊n/2⌋ leading zeros. For 011, n=2 and ⌊2/2⌋=1; for 0111, n=3 and ⌊3/2⌋=1; for 001111, n=4 and ⌊4/2⌋=2. All three fit. The string 0011 has n=2 ones but two zeros, whereas the rule permits only one zero, so it is not in A.

## Key Points

- For a parameterized language, infer n from a candidate string and then verify every exponent constraint.

## Why the other options are incorrect

Options 1, 2, and 4 are generated by n=2, n=3, and n=4 respectively. Option 3 cannot be repaired by choosing a different n because n is fixed by the number of trailing ones in the candidate string.
