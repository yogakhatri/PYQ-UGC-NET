# Question 35

*UGC NET CS · 2010 June Paper 2 · Code Generation and Optimization · Control-Flow and Data-Flow Analysis*

Which of the following is used for grouping of characters into tokens (in a computer) ?

- **A.** A parser
- **B.** Code optimizer
- **C.** Code generator
- **D.** Scanner

> [!TIP]
> **Correct answer: D. Scanner**

## Solution

The lexical analyzer, also called a scanner, reads the character stream and groups characters into lexemes, emitting tokens such as identifiers, keywords, and numeric literals.

## Key Points

- Characters→scanner→tokens→parser→syntax structure.

## Why the other options are incorrect

A parser groups tokens into grammatical structures; optimization and code generation occur later and do not perform lexical grouping.
