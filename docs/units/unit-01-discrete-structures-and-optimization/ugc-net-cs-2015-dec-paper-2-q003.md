# Question 3

*UGC NET CS · 2015 Dec Paper 2 · Sets and Relations · Countable and Uncountable Sets*

Which of the following are not true? (a) The set of negative integers is countable. (b) The set of integers that are multiples of 7 is countable. (c) The set of even integers is countable. (d) The set of real numbers between 0 and 1/2 is countable.

- **1.** (a) and (c)
- **2.** (b) and (d)
- **3.** (b) only
- **4.** (d) only

> [!TIP]
> **Correct answer: 4. (d) only**

## Solution

The negative integers, multiples of 7, and even integers are all countably infinite: list them as −1,−2,…; 0,7,−7,14,−14,…; and 0,2,−2,4,−4,… respectively. The open interval (0,1/2) is uncountable because scaling x ↦ 2x gives a bijection from it to (0,1), which has the cardinality of the real continuum. Hence only statement (d) is not true.

## Key Points

- Infinite subsets of the integers can be countable, but every nonempty real interval is uncountable.

## Why the other options are incorrect

Options 1–3 incorrectly classify one of the integer subsets as uncountable. Every infinite subset listed in (a)–(c) can be placed in a one-to-one enumeration with the natural numbers.
