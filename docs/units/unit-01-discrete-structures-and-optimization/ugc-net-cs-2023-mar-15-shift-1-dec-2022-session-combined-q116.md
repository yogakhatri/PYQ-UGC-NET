# Question 116

*UGC NET CS · 2023 Mar 15 Shift 1 Dec 2022 Session · Mathematical Logic · Propositional and Predicate Logic*

Match List I with List II List I List II A. Linear Program l. Prim's Algorithm Il. In order B. Predicate Logic C. Tree Traversal III. Simplex Algorithm D. Minimum Spanning Tree Problem IV. First order logic Choose the correct answer from the options given below:

- **1.** A-IV; B-III; C-I; D-II
- **2.** A-III; B-IV; C-Il; D-I
- **3.** A-II; B-IV; C-I; D-III
- **4.** A-III; B-IV; C-I; D-II

> [!TIP]
> **Correct answer: 2. A-III; B-IV; C-Il; D-I**

## Solution

Linear programs use the simplex algorithm (A–III); predicate logic is first-order logic (B–IV); inorder is a tree traversal (C–II); Prim solves MST (D–I).

## Key Points

- Simplex→LP, Prim→MST, inorder→tree, predicate→first-order logic.

## Why the other options are incorrect

Other mappings swap algorithms with unrelated problem classes.
