# Question 91

*UGC NET CS · 2023 Mar 15 Shift 1 Dec 2022 Session · Group Theory · Products and Quotients of Algebraic Structures*

l. No.4 BID:8704: Consider the following statements: A. A group can have zero elements. B. A group cannot have any element which is idempotent except the identity element. C. An isomorphism from a group < G, *> to < G, *> is called an automorphism. D. If every element in a group is its own inverse, then the group must be abelian. Choose the correct answer about the four statements given above.

- **1.** B, C and D are correct
- **2.** Only A and Care correct
- **3.** Only B and C are correct
- **4.** All A, B, C and D are correct.

> [!TIP]
> **Correct answer: 1. B, C and D are correct**

## Solution

A group cannot be empty because it must contain an identity, so A is false. If g²=g, cancellation gives g=e, so only the identity is idempotent (B). An isomorphism from a group to itself is an automorphism (C). If every element is self-inverse, (ab)⁻¹=ab but also b⁻¹a⁻¹=ba, hence ab=ba (D).

## Key Points

- A group has an identity; exponent-two groups are abelian.

## Why the other options are incorrect

Options 2 and 4 accept false A; option 3 omits true D.
