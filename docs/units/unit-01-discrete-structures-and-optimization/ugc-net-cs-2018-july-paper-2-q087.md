# Question 87

*UGC NET CS · 2018 July Paper 2 · Functions · Injective, Surjective and Constant Functions*

Match the properties of a function f: (a) ∀x∀y(f(x)=f(y)→x=y), (b) ∀y∃x(f(x)=y), (c) ∀x f(x)=k; with (i) constant, (ii) injective, and (iii) surjective.

- **1.** (i) (ii) (iii)
- **2.** (iii) (ii) (i)
- **3.** (ii) (i) (iii)
- **4.** (ii) (iii) (i)

> [!TIP]
> **Correct answer: 4. (ii) (iii) (i)**

## Solution

Condition (a), f(x)=f(y)⇒x=y, is the definition of injective, so a→ii. Condition (b), every codomain value y has some preimage x with f(x)=y, is surjective, so b→iii. Condition (c), every x maps to the same k, is constant, so c→i. The sequence (ii),(iii),(i) is option 4.

## Key Points

- Injective: equal outputs imply equal inputs; surjective: every codomain value is hit; constant: every input has one fixed output.

## Why the other options are incorrect

The other matchings confuse one-to-one with onto or constant behavior. Remember: injective constrains pairs of inputs, while surjective quantifies over every target value.
