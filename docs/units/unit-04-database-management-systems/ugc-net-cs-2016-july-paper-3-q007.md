# Question 7

*UGC NET CS · 2016 July Paper 3 · Object-Oriented Databases · Object and Object-Relational Databases*

Which of the following statements concerning Object-Oriented databases is FALSE ?

- **1.** Objects in an object-oriented database contain not only data but also methods for processing the data.
- **2.** Object-oriented databases store computational instructions in the same place as the data.
- **3.** Object-oriented databases are more adept at handling structured (analytical) data than relational databases.
- **4.** Object-oriented databases store more types of data than relational databases and access that data faster.

> [!TIP]
> **Correct answer: 3. Object-oriented databases are more adept at handling structured (analytical) data than relational databases.**

## Solution

Statement 3 is false. Relational databases are especially strong for regular structured data and analytical queries expressed through tables, joins, grouping, and aggregation. Object-oriented databases are chiefly advantageous for complex object graphs, user-defined types, multimedia, and close correspondence with object-oriented programs.

## Key Points

- Relational DBMSs excel at structured tabular analytics; object DBMSs preserve complex objects, identity, methods, and relationships.

## Why the other options are incorrect

Statements 1 and 2 describe encapsulation of state and behavior in persistent objects. Statement 4 expresses the intended advantage that object databases can store richer object types and avoid object-relational conversion overhead. The comparative claim in statement 3 reverses the usual strength of relational systems.
