# Question 58

*UGC NET CS · 2015 Dec Paper 3 · Normalization for Relational Databases · Object and Object-Relational Databases*

Which of the following statements regarding the features of the object-oriented approach to databases are true ? (a) The ability to develop more realistic models of the real world. (b) The ability to represent the world in a non-geometric way. (c) The ability to develop databases using natural language approaches. (d) The need to split objects into their component parts. (e) The ability to develop database models based on location rather than state and behaviour. Codes :

- **1.** (a), (b) and (c)
- **2.** (b), (c) and (d)
- **3.** (a), (d) and (e)
- **4.** (c), (d) and (e)

> [!TIP]
> **Correct answer: 1. (a), (b) and (c)**

## Solution

The object-oriented database approach can model real-world entities directly as objects with identity, complex state, behavior, inheritance, and relationships. This supports more realistic models (a), representations beyond purely geometric/spatial structures (b), and high-level or natural-language-oriented conceptual development (c). Hence option 1 is the intended set.

## Key Points

- Object databases preserve complex objects, identity, state, behavior, and inheritance close to the application model.

## Why the other options are incorrect

Statement (d) reverses an OO advantage: an object can be stored as an integrated complex object rather than being forcibly decomposed across flat relations. Statement (e) is also contrary to OO modeling, which centers on object identity, state, and behavior rather than location alone. Every other code includes (d) or (e).
