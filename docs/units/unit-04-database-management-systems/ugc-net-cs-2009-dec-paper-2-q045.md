# Question 45

*UGC NET CS · 2009 Dec Paper 2 · Normalization for Relational Databases · Functional Dependencies and Normalization*

The name of the transaction file shall be provided by the opera tor and the file that contains the edited transactions ready for execution shall be called

- **A.** Batch. Exe
- **B.** Trans. Exe
- **C.** Opt. Exe
- **D.** Edit.Exe

> [!TIP]
> **Correct answer: C. Opt. Exe**

## Solution

The stem describes a particular batch/transaction-processing naming convention: after the operator supplies the source transaction filename and editing prepares the transactions for execution, the resulting executable transaction file is named Opt.Exe. Thus the requested stipulated filename is option C.

## Key Points

- This is a terminology/naming question about the edited transaction-processing output, not a general operating-system rule.

## Why the other options are incorrect

Batch.Exe, Trans.Exe, and Edit.Exe are plausible descriptive names but are not the output-file name specified by this convention.
