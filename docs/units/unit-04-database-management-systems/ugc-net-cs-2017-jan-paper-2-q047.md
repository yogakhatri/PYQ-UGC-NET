# Question 47

*UGC NET CS · 2017 Jan Paper 2 · Data Warehousing and Data Mining · Definition and Purpose of Data Mining*

________ refers loosely to the process of semi-automatically analyzing large databases to find useful patterns.

- **1.** Data mining
- **2.** Data warehousing
- **3.** DBMS
- **4.** Data mirroring

> [!TIP]
> **Correct answer: 1. Data mining**

## Solution

Data mining is the semi-automatic or computational discovery of useful, previously unknown patterns, associations, clusters, anomalies, or predictive relationships in large datasets. That definition matches the stem, so option 1 is correct.

## Key Points

- Warehouse stores analysis-ready data; data mining discovers patterns in it.

## Why the other options are incorrect

A data warehouse integrates historical data for analysis but is a repository and architecture, not the pattern-discovery process itself. A DBMS manages storage and queries generally. Data mirroring maintains a duplicate copy for availability or recovery.
