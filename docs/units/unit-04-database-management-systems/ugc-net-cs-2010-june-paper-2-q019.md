# Question 19

*UGC NET CS · 2010 June Paper 2 · Enhanced Data Models · Genome Data Management*

Which data management language component enabled the DBA to define the schema components ?

- **A.** DML
- **B.** Sub-schema DLL
- **C.** Schema DLL
- **D.** All of these

> [!TIP]
> **Correct answer: C. Schema DLL**

## Solution

The schema data-definition language (schema DDL) lets the database administrator define the conceptual schema: relations/entities, attributes, constraints, and related structural components.

## Key Points

- DDL defines structure; DML operates on stored data.

## Why the other options are incorrect

DML manipulates/query data. A subschema DDL defines user/application views rather than the overall database schema, so 'all' is false.
