# Question 50

*UGC NET CS · 2016 July Paper 2 · Data Warehousing and Data Mining · Temporal Data Value and Expert-System Knowledge Acquisition*

Given the following statements : (A) Strategic value of data mining is timestamping. (B) Information collection is an expensive process in building an expert system. Which of the following options is correct ?

- **1.** Both (A) and (B) are false.
- **2.** Both (A) and (B) are true.
- **3.** (A) is true, (B) is false.
- **4.** (A) is false, (B) is true.

> [!TIP]
> **Correct answer: 2. Both (A) and (B) are true.**

## Solution

Statement A is treated as true in this item: timestamping attaches time context to records or events, enabling temporal ordering, trend analysis, sequence discovery, freshness checks, and time-sensitive decisions—valuable uses of mined data. Statement B is true because acquiring, eliciting, validating, and encoding expert knowledge is labor-intensive and is a major cost in building an expert system. Therefore both statements are true, option 2.

## Key Points

- Temporal metadata makes time-based mining possible; expert-system knowledge acquisition is costly and time-consuming.

## Why the other options are incorrect

Options 1, 3, and 4 each mark at least one of the two accepted statements false. Information collection is not a trivial programming step: scarce expert time and repeated validation make knowledge acquisition a well-known bottleneck.

## Additional Information

Statement A is awkwardly phrased. Timestamping is not a complete definition of the strategic value of data mining; it is one mechanism that preserves temporal context and supports strategically useful temporal analysis. Read the statement in that narrower exam-intended sense.
