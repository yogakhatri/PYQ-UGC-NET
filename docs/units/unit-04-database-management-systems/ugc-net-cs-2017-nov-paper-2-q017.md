# Question 17

*UGC NET CS · 2017 Nov Paper 2 · SQL · Aggregate Functions*

In SQL, __________ is an Aggregate function.

- **1.** SELECT
- **2.** CREATE
- **3.** AVG
- **4.** MODIFY

> [!TIP]
> **Correct answer: 3. AVG**

## Solution

`AVG` is an SQL aggregate function: it takes the non-NULL values of an expression across a group of rows and returns their arithmetic mean. Therefore option 3 is correct.

## Key Points

- Aggregate functions summarize many input rows into one value per group (or one value for the entire result when there is no grouping).

## Why the other options are incorrect

`SELECT` is a query statement, `CREATE` is a data-definition command, and `MODIFY` is not an aggregate function. Other common aggregates are `COUNT`, `SUM`, `MIN`, and `MAX`.
