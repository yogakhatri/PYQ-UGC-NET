# Question 119

*UGC NET CS · 2023 Mar 11 Shift 2 Dec 2022 Session · Big Data Systems · Big-data ecosystem tools*

Match List I and List II LIST T A. Apache Kafka Platform for constructing data flows for extract, transform, and load (ETL) processing and analysis of large darasets. B. Pig II. General-purpose computing model and runtime system for distributed data analytics. Apache Mahout III. Open-source platform that was created by Linkedin. Mapreduce IV. Open-source platform used for creating scalable machine learning algorithms. Choose the correct answer from the options given below:

- **1.** A-II, B-IV, C-I, D-III
- **2.** A-II, B-I, C-IV, D-III
- **3.** A-II, B-I, C-IV, D-II
- **4.** A-I, B-II, C-IV, D-III

> [!TIP]
> **Correct answer: 3. A-II, B-I, C-IV, D-II**

## Solution

Kafka was created at LinkedIn (A–III); Pig supplies data-flow/ETL processing (B–I); Mahout supplies scalable machine-learning algorithms (C–IV); MapReduce is a distributed computation model/runtime (D–II).

## Key Points

- Kafka: streaming; Pig: ETL; Mahout: ML; MapReduce: distributed computation.

## Why the other options are incorrect

Other mappings interchange platforms with distinct origins and purposes.
