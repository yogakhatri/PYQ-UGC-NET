# Question 88

*UGC NET CS · 2019 June Paper 1 And 2 · Big Data Systems · Hadoop ecosystem*

Hadoop (a big data tool) works with number of related tools. Choose from the following, the common tools included into Hadoop :

- **1.** MySQL, Google API and Map reduce
- **2.** Map reduce, Scala and Hummer
- **3.** Map reduce, H Base and Hive
- **4.** Map reduce, Hummer and Heron

> [!TIP]
> **Correct answer: 3. Map reduce, H Base and Hive**

## Solution

MapReduce is Hadoop's distributed processing model, HBase is a distributed column-family database built on the Hadoop ecosystem, and Hive provides SQL-like data warehousing and querying over distributed data. These three are established Hadoop-related tools.

## Key Points

- Common Hadoop ecosystem names include HDFS, MapReduce, YARN, HBase and Hive.

## Why the other options are incorrect

MySQL and the generic Google API are not Hadoop components. Hummer and Heron are not the standard tools named in the Hadoop stack represented by this question.
