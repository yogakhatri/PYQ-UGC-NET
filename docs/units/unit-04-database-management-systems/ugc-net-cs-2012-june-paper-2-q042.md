# Question 42

*UGC NET CS · 2012 June Paper 2 · Data Warehousing and Data Mining · E-Commerce Data Warehousing*

Identify the incorrect statement :

- **A.** The overall strategy drives the E-Commerce data warehousing strategy.
- **B.** Data warehousing in an E-Commerce environment should be done in a classical manner.
- **C.** E-Commerce opens up an entirely new world of web server.
- **D.** E-Commerce security threats can be grouped into three major categories. www.solutionsadda.in

> [!TIP]
> **Correct answer: B. Data warehousing in an E-Commerce environment should be done in a classical manner.**

## Solution

E-commerce produces clickstream events, rapidly changing products, customer interactions, external campaign data and often near-real-time analytical needs. A warehouse strategy must adapt its sources, refresh, scalability, identity resolution and security to this environment. Therefore saying it should be done in a purely classical manner is the incorrect statement.

## Key Points

- An e-commerce warehouse extends classical warehousing for web-scale, behavioural and faster-changing data.

## Why the other options are incorrect

Overall business strategy should drive warehouse strategy. E-commerce creates new web-server and interaction data to analyse. Its security threats can reasonably be grouped into broad categories such as client, communication-channel and server threats. Those statements are not the incorrect one sought.
