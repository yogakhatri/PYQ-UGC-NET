# Question 87

*UGC NET CS · 2019 June Paper 1 And 2 · SQL · Data manipulation statements*

Which statements are data-manipulation language (DML) statements? (a) UPDATE table SET column=value. (b) DELETE FROM table. (c) SELECT * FROM table.

- **1.** (a) and (b) only
- **2.** (a) only
- **3.** (a), (b) and (c)
- **4.** (b) and (c) only

> [!TIP]
> **Correct answer: 3. (a), (b) and (c)**

## Solution

UPDATE modifies existing rows, DELETE removes rows and SELECT retrieves rows. In the standard SQL command classification used in database courses, all three operate on table data and are treated as DML statements. Therefore (a), (b) and (c) are included.

## Key Points

- Core SQL DML commands are SELECT, INSERT, UPDATE and DELETE.

## Why the other options are incorrect

Options 1, 2 and 4 each omit at least one of UPDATE, DELETE or SELECT even though all three manipulate or query stored data.
