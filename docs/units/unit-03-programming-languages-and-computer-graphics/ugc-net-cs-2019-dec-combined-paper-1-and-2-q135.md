# Question 135

*UGC NET CS · 2019 Dec Paper 1 And 2 · Web Programming · HTML Tables and Links*

Match List I with List II. List I: A. frame attribute; B. <tr> tag; C. valign attribute; D. <a> tag. List II: I. creates a link in HTML; II. specifies vertical alignment of content in a cell; III. encloses a row in a table; IV. specifies which sides of a table frame display a border.

- **1.** A-I, B-II, C-III, D-IV
- **2.** A-II, B-I, C-III, D-IV
- **3.** A-IV, B-III, C-II, D-I
- **4.** A-III, B-IV, C-II, D-I

> [!TIP]
> **Correct answer: 3. A-IV, B-III, C-II, D-I**

## Solution

The legacy HTML table frame attribute controls which outer sides of a table's frame are rendered, giving A-IV. The <tr> element denotes and encloses a table row, giving B-III. The valign attribute specifies vertical alignment of content inside a table cell or row, giving C-II. The anchor element <a> creates a hyperlink, giving D-I. Therefore the correct mapping is A-IV, B-III, C-II, D-I—option 3.

## Key Points

- HTML roles: frame=outer border sides, <tr>=row, valign=vertical alignment, <a>=link.

## Why the other options are incorrect

Options 1, 2, and 4 misassign at least the frame or row role. Do not confuse frame, which selects outer table-border sides, with <tr>, which supplies row structure, or valign, which changes vertical placement rather than table structure.
