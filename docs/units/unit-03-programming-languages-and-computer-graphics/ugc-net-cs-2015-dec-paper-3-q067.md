# Question 67

*UGC NET CS · 2015 Dec Paper 3 · Display Devices · Memory-Tube Display Components*

Which listed items are principal components of a memory-tube display? (a) Flooding gun, (b) Collector, (c) Phosphor grains, (d) Ground.

- **1.** (a) and (b)
- **2.** (c) only
- **3.** (d) only
- **4.** All of the above

> [!TIP]
> **Correct answer: 4. All of the above**

## Solution

A memory-tube/storage display uses a flood gun to supply a broad low-energy electron stream. Its storage target includes a collector structure and phosphor or dielectric grains associated with a grounded conductive backing; the stored charge pattern controls where the flood electrons reach the display. Thus all four listed elements belong to the textbook memory-tube construction.

## Key Points

- A storage display maintains a charge pattern that modulates flood electrons, avoiding continual redraw from refresh memory.

## Why the other options are incorrect

Options 1–3 each select only part of the assembly. The display needs an electron source, charge-storage/collection structure, luminescent target material, and an electrical reference or return path, so none of the partial combinations is complete.
