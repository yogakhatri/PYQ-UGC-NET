# Question 1

*UGC NET CS · 2021 Nov With Answers · Web Programming · HTML Image Alternative Text and Accessibility*

Which of the statements given below is/are correct? It is always important and useful to include an 'alt' attribute on 'img' tag in HTML because A. users who cannot see the image due to vision impairment can have a textual description of the image (which can be spoken aloud by a screenreader). B. If the image fails to load (slow connection, broken path, etc.) then alt text is displayed instead. C. SEO (Search Engine Optimization) benefits. Choose the correct answer from the options given below:

- **1.** A only
- **2.** B and C only
- **3.** C only
- **4.** A, B, and C

> [!TIP]
> **Correct answer: 4. A, B, and C**

## Solution

Alternative text gives a nonvisual replacement for meaningful image content, so a screen reader can communicate the image's purpose to a user who cannot see it; A is correct. Browsers can also display alt text when the image resource cannot be loaded, so B is correct. Descriptive alternative text helps search engines understand the image and its surrounding page context, so C is also a legitimate SEO benefit. Therefore A, B, and C are correct: option 4.

## Key Points

- The alt attribute supplies a text alternative for accessibility, loading failure, and machine understanding of meaningful images.

## Why the other options are incorrect

Options 1–3 omit at least one valid purpose. Alt text should describe the image's function or content rather than be stuffed with keywords; decorative images normally use an empty alt attribute, but the attribute itself should still be present.
