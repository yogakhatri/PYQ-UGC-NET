# Question 13

*UGC NET CS · 2014 Dec Paper 3 · Animation · Categories of Animation Languages*

Which of the following categories of languages do not refer to animation languages ?

- **A.** Graphical languages
- **B.** General-purpose languages
- **C.** Linear-list notations
- **D.** None of the above

> [!TIP]
> **Correct answer: D. None of the above**

## Solution

Animation systems have historically been specified through graphical languages, general-purpose programming languages and linear-list/script notations. Each of A–C therefore refers to a valid category of animation language, so none is the category that 'does not refer.'

## Key Points

- Animation specification may be graphical, general-purpose/programmatic, or linear-list/script based.

## Why the other options are incorrect

A provides visual or diagrammatic animation specification. B implements animation with ordinary languages and libraries. C represents time-ordered actions in textual lists. Since all three apply, D is correct.
