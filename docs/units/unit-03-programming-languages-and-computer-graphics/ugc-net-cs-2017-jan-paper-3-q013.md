# Question 13

*UGC NET CS · 2017 Jan Paper 3 · Computer Graphics · CRT Components and Phosphor Persistence*

Which of the following statement(s) is/are correct ?

- **1.** Persistence is the term used to describe the duration of phosphorescence.
- **2.** The control electrode is used to turn the electron beam on and off.
- **3.** The electron gun creates a source of electrons which are focussed into a narrow beam directed at the face of CRT.
- **4.** All of the above

> [!TIP]
> **Correct answer: 4. All of the above**

## Solution

All three CRT statements are correct. Persistence is the time for emitted phosphor light to decay after excitation. The control grid/electrode controls electron-beam current and can blank the beam. The electron gun emits electrons and the focusing system forms them into a narrow beam aimed at the phosphor-coated screen. Therefore 'All of the above,' option 4, is correct.

## Key Points

- CRT chain: electron gun creates the beam, control grid modulates it, focusing directs it, and phosphor persistence determines glow decay.

## Why the other options are incorrect

Options 1–3 each state only one true fact. Because none of those statements is false, the inclusive choice covering all three is required.
