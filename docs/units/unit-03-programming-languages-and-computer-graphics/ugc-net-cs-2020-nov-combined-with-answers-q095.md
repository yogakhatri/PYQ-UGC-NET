# Question 95

*UGC NET CS · 2020 Nov With Answers · Web Programming · XML Syntax, Semantics and Case Sensitivity*

Which statements regarding XML are true? (A) XML is a set of tags designed to tell browsers how to display text and images on a web page. (B) XML defines a syntax for representing data, but the meaning of the data varies from application to application. (C) <Letter>, <LETTER>, and <letter> are three different tags in XML.

- **1.** (A) and (B) only
- **2.** (A) and (C) only
- **3.** (B) and (C) only
- **4.** (A), (B) and (C)

> [!TIP]
> **Correct answer: 3. (B) and (C) only**

## Solution

XML provides a general syntax for structuring and exchanging data; an application or vocabulary assigns meaning to the element names, so B is true. XML names are case-sensitive, making <Letter>, <LETTER>, and <letter> three distinct tags, so C is true. Statement A is false because it describes display-oriented markup such as HTML; XML tags are user-defined and primarily express data structure, not fixed browser presentation. Therefore B and C only, option 3.

## Key Points

- XML is extensible, application-semantic, and case-sensitive; it structures data rather than prescribing page display.

## Why the other options are incorrect

Options 1, 2, and 4 include A. A browser may display or transform XML with additional styling rules, but XML itself is not a predefined collection of presentation tags.
