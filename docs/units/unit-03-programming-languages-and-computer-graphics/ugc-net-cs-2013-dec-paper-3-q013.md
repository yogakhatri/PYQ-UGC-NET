# Question 13

*UGC NET CS · 2013 Dec Paper 3 · Web Programming · XML and XSLT Stylesheets*

The document that is used by XSLT to indicate, how to transform the elements of the XML document to another format is

- **A.** HTML page
- **B.** DOC type procedure
- **C.** Style sheet
- **D.** Stored procedure

> [!TIP]
> **Correct answer: C. Style sheet**

## Solution

XSLT transformations are defined in an XSLT stylesheet. The stylesheet contains templates that match XML nodes and specifies the result structure, which may be XML, HTML or text.

## Key Points

- XML data + XSLT stylesheet → transformed XML, HTML or text output.

## Why the other options are incorrect

An HTML page can be an output but does not direct XSLT. A document type declaration describes allowed XML structure, and a stored procedure is database code. Neither contains XSLT transformation templates.
