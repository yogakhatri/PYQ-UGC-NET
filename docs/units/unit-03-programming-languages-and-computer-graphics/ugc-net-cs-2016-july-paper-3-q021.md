# Question 21

*UGC NET CS · 2016 July Paper 3 · Language Design and Translation Issues · Programming-language evaluation criteria*

Which of the following is false regarding the evaluation of computer programming languages ?

- **1.** Application oriented features
- **2.** Efficiency and Readability
- **3.** Software development
- **4.** Hardware maintenance cost

> [!TIP]
> **Correct answer: 4. Hardware maintenance cost**

## Solution

Programming languages are evaluated using criteria such as suitability for application domains, readability, writability, reliability, implementation/execution efficiency, and the effect on software development and maintenance. Hardware maintenance cost is not a property or standard evaluation criterion of a programming language. Therefore option 4 is false.

## Key Points

- Language evaluation concerns program clarity, reliability, cost, efficiency, and domain suitability—not maintenance of physical hardware.

## Why the other options are incorrect

Application-oriented features determine fitness for a problem domain. Efficiency and readability are conventional language-quality criteria. Software-development effort and maintainability are also strongly influenced by language design. Those three therefore belong in language evaluation.
