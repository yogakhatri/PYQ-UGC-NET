# Question 14

*UGC NET CS · 2013 Dec Paper 3 · Language Implementation and Runtime Systems · Dynamic Loading*

Which of the following concepts means adding new concepts to a program as it runs ?

- **A.** Data hiding
- **B.** Dynamic loading
- **C.** Dynamic typing
- **D.** Dynamic binding

> [!TIP]
> **Correct answer: B. Dynamic loading**

## Solution

Dynamic loading brings a code module or class into a running program only when it is needed. This lets the application acquire functionality without linking and loading every component at startup.

## Key Points

- Dynamic loading adds code at runtime; dynamic binding chooses behavior at runtime; dynamic typing determines types at runtime.

## Why the other options are incorrect

Data hiding restricts access to representation, dynamic typing determines/checks types at runtime, and dynamic binding selects an overridden method at runtime. None specifically adds program code during execution.
