# Question 34

*UGC NET CS · 2015 Dec Paper 3 · Web Programming · JavaScript versus Java*

Which statements about JavaScript and Java are true? (a) JavaScript's syntax is loosely based on Java's syntax. (b) JavaScript is a stripped-down Java. (c) Both originated from the island of Java.

- **1.** (a) only
- **2.** (a), (b) and (c)
- **3.** (a) and (b)
- **4.** (a) and (c)

> [!TIP]
> **Correct answer: 1. (a) only**

## Solution

JavaScript deliberately adopted Java-like surface syntax for familiar constructs such as braces, expressions, and control statements, so (a) is true. It was nevertheless designed as a separate dynamic language and is not a stripped-down Java. Neither language originated from the Indonesian island; the names were branding choices. Therefore only (a) is true.

## Key Points

- JavaScript and Java are distinct languages; their strongest connection here is superficial syntax and naming.

## Why the other options are incorrect

Every other code includes false statement (b), (c), or both. Similar syntax and similar branding do not imply a shared implementation or direct subset relationship.
