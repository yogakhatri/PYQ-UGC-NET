# Question 4

*UGC NET CS · 2021 Nov With Answers · Java Programming · JNDI, RMI, JDK and JRE*

Match List I with List II List I List II (Programming Term) (Meaning) A. JNDI I. Runtime support for running Java programs B. RMI II. The API in support of naming and directory services C. JDK III. The methods provided by the Java development kit and runtime support for calling remote methods D. JRE IV. The compiler and class libraries to develop Java applications Choose the correct answer from the options given below:

- **1.** A ‐ II , B ‐ III, C ‐ IV, D ‐ I
- **2.** A ‐ II, B ‐ IV, C ‐ III, D ‐ I
- **3.** A ‐ I, B ‐ III, C ‐ IV, D ‐ II
- **4.** A ‐ III, B ‐ II, C ‐ IV, D ‐ I

> [!TIP]
> **Correct answer: 1. A ‐ II , B ‐ III, C ‐ IV, D ‐ I**

## Solution

JNDI is the Java Naming and Directory Interface, the API for naming and directory services, so A→II. RMI is Remote Method Invocation, providing support for calls on remote objects, so B→III. The JDK contains development tools such as the compiler plus class libraries, so C→IV. The JRE supplies the runtime needed to execute Java programs, so D→I. The mapping is option 1.

## Key Points

- JNDI names, RMI invokes remotely, JDK develops, and JRE runs.

## Why the other options are incorrect

The other mappings confuse the development kit with the runtime environment or assign naming services to RMI. The acronym expansions make each match direct.
