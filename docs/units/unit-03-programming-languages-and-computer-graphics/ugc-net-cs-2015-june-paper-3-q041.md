# Question 41

*UGC NET CS · 2015 June Paper 3 · Web Programming · XML DOCTYPE and DTD*

In XML, DOCTYPE declaration specifies to include a reference to __________ file.

- **1.** Document type Definition
- **2.** Document type declaration
- **3.** Document transfer definition
- **4.** Document type language

> [!TIP]
> **Correct answer: 1. Document type Definition**

## Solution

An XML `DOCTYPE` declaration can contain an internal subset or reference an external DTD. DTD expands to Document Type Definition and declares the permitted elements, attributes, entities, and structural relationships for that document type. Hence option 1 is correct.

## Key Points

- DOCTYPE points to or contains a DTD—Document Type Definition.

## Why the other options are incorrect

The other expansions are not XML standard terms. `DOCTYPE` is the declaration in the XML document; the referenced rules file is the Document Type Definition.
