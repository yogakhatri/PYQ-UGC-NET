# Question 41

*UGC NET CS · 2016 July Paper 3 · Web Programming · DHTML*

Which of the following statements is not correct ?

- **1.** HTML is not screen precise formatting language.
- **2.** HTML does not specify a logic.
- **3.** DHTML is used for developing highly interactive web pages.
- **4.** HTML is a programming language.

> [!TIP]
> **Correct answer: 4. HTML is a programming language.**

## Solution

HTML is a markup language: tags describe document structure and semantics for a user agent. It is not, by itself, a programming language with variables, control flow, and computational logic. Therefore statement 4 is not correct.

## Key Points

- HTML structures content; CSS styles it; JavaScript supplies programmable behavior; DHTML refers to their dynamic combination.

## Why the other options are incorrect

HTML does not guarantee pixel-perfect screen formatting because rendering depends on CSS, browser, viewport, and fonts. HTML alone does not specify program logic. DHTML is the historical name for combining HTML, CSS, DOM, and scripting to make pages interactive. These statements are correct.
