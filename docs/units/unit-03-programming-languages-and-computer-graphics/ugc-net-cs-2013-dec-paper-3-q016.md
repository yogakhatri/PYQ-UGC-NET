# Question 16

*UGC NET CS · 2013 Dec Paper 3 · Programming-Language Paradigms · Structured, Object-Oriented and Interpreted Languages*

Match the following with respect to programming languages : List – I List – II a. Structured Language i. JAVA b. Non-structured Language ii. BASIC c. Object Oriented Programming Language iii. PASCAL d. Interpreted Programming Language iv. FORTRAN Codes : a b c d

- **A.** iii iv i ii
- **B.** iv iii ii i
- **C.** ii iv i iii
- **D.** ii iii iv i

> [!TIP]
> **Correct answer: A. iii iv i ii**

## Solution

Pascal is the structured-language example (a-iii). Traditional FORTRAN is the non-structured example (b-iv). Java is object-oriented (c-i), and classic BASIC is commonly interpreted (d-ii). The complete code is iii,iv,i,ii.

## Key Points

- Exam-era associations: Pascal-structured, FORTRAN-non-structured, Java-object-oriented, BASIC-interpreted.

## Why the other options are incorrect

The other codes assign at least one language to the wrong paradigm. Java→object-oriented and BASIC→interpreted are strong anchors; Pascal and early FORTRAN then distinguish structured from non-structured programming.
