# Question 41

*UGC NET CS · 2013 Dec Paper 2 · Web Programming · ActiveX Controls*

Active X controls are Pentium binary programs that can be embedded in ________

- **A.** Word pages
- **B.** URL pages
- **C.** Script pages
- **D.** Web pages

> [!TIP]
> **Correct answer: D. Web pages**

## Solution

ActiveX controls were compiled Windows binary components that Internet Explorer and related container applications could embed in web pages to provide interactive functionality. They executed native code on the client machine, which also created substantial security risk.

## Key Points

- ActiveX was native Windows code embedded in web pages, unlike portable sandboxed web scripting; it is now obsolete and unsafe for modern web design.

## Why the other options are incorrect

A URL is an address, not a page container, and 'script page' is not the defining host. Word can host some embedded objects, but the exam statement specifically describes the historical use of ActiveX controls in web pages.
