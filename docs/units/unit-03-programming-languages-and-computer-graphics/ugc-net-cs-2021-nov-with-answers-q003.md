# Question 3

*UGC NET CS · 2021 Nov With Answers · Programming Concepts · Tracing Recursive String Construction*

Consider the Java function public static String F(int N) { if (N<=0) return "-"; return F(N-3)+N+F(N-2)+N; }. What is F(5)?

- **1.** ‐2‐25‐3‐135
- **2.** ‐2‐25‐1‐3‐135
- **3.** ‐1‐145‐2‐245
- **4.** ‐2‐25‐3‐1‐135

> [!TIP]
> **Correct answer: 4. ‐2‐25‐3‐1‐135**

## Solution

Build only the needed smaller results. F(2)=F(−1)+2+F(0)+2="-2-2". Also F(1)="-1-1", so F(3)=F(0)+3+F(1)+3="-3-1-13". Therefore F(5)=F(2)+5+F(3)+5="-2-25-3-1-135". This matches option 4.

## Key Points

- For recursive string functions, compute a small call table and preserve exact left-to-right concatenation order.

## Why the other options are incorrect

The alternatives lose, reorder, or duplicate pieces generated by the two recursive calls. String + performs concatenation from left to right; it does not numerically add the displayed integers.
