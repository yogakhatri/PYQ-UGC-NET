# Question 17

*UGC NET CS · 2013 Dec Paper 3 · Elementary Data Types · Type Promotion*

The compiler converts all operands upto the type of the largest operand is called

- **A.** Type Promotion
- **B.** Type Evaluation
- **C.** Type Conversion
- **D.** Type Declaration

> [!TIP]
> **Correct answer: A. Type Promotion**

## Solution

When an expression contains operands of different ranks, the language's usual arithmetic rules convert narrower operands toward a wider common type so the operation can be performed consistently. This automatic upward conversion is type promotion.

## Key Points

- Promotion is an implicit widening conversion used to bring operands to a common higher-ranked type.

## Why the other options are incorrect

Type conversion is a broad term covering many explicit and implicit changes, but promotion specifically means conversion to a higher-ranked type. Evaluation computes an expression, and declaration introduces a typed name.
