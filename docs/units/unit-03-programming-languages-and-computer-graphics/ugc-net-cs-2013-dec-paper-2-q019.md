# Question 19

*UGC NET CS · 2013 Dec Paper 2 · Programming in C++ · Inheritance*

_____ allows creation of classes derived from other classes so that they automatically include some parent members plus their own members.

- **A.** Overloading
- **B.** Inheritance
- **C.** Polymorphism
- **D.** Encapsulation

> [!TIP]
> **Correct answer: B. Inheritance**

## Solution

Inheritance defines a new derived class from an existing base class. The derived class receives accessible base-class data and behavior and may add or specialize members of its own. It supports reuse and models an appropriate 'is-a' relationship.

## Key Points

- Inheritance creates derived classes from base classes; use it for genuine is-a relationships.

## Why the other options are incorrect

Overloading gives the same name to different operations, polymorphism allows one interface to select different implementations, and encapsulation bundles state with operations while controlling access. None specifically creates a class from a parent class.
