# Question 75

*UGC NET CS · 2015 Dec Paper 3 · Web Programming · JavaScript SCRIPT Element*

__________ tag is an extension to HTML that can enclose any number of Javascript statements.

- **1.** <SCRIPT>
- **2.** <BODY>
- **3.** <HEAD>
- **4.** <TITLE>

> [!TIP]
> **Correct answer: 1. <SCRIPT>**

## Solution

The HTML <script> element embeds or references executable script code. When used inline, its start and end tags can enclose any number of JavaScript statements, so option 1 is correct.

## Key Points

- Use <script>...</script> for inline JavaScript; use its src attribute for an external script.

## Why the other options are incorrect

<body> contains the document's rendered content, <head> contains metadata and linked resources, and <title> supplies the document title. A <script> element may appear in the head or body, but those container elements do not themselves identify enclosed text as JavaScript.
