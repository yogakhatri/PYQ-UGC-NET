# Question 66

*UGC NET CS · 2019 Dec Paper 1 And 2 · Web Programming · HTML*

Which tag is used to enclose any number of javascript statements in HTML document?

- **1.** <code>
- **2.** <script>
- **3.** <title>
- **4.** <body>

> [!TIP]
> **Correct answer: 2. <script>**

## Solution

HTML uses the <script> element to embed executable scripting code, including any sequence of JavaScript statements, or to reference an external script file. Therefore option 2 is correct.

## Key Points

- Use <script> for executable client-side script; use <code> only for code displayed as content.

## Why the other options are incorrect

<code> marks text as a code fragment but does not execute it. <title> supplies the document title shown by the browser, and <body> contains the page's rendered content; neither specifically encloses executable JavaScript.
