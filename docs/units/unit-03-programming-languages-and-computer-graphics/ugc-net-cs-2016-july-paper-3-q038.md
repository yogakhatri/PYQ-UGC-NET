# Question 38

*UGC NET CS · 2016 July Paper 3 · Object-Oriented Programming · Static Nested and Inner Classes*

It is possible to define a class within a class termed as nested class. There are _____ types of nested classes.

- **1.** 2
- **2.** 3
- **3.** 4
- **4.** 5

> [!TIP]
> **Correct answer: 1. 2**

## Solution

At the top level, Java divides nested classes into two categories: static nested classes and non-static nested classes, the latter called inner classes. Therefore the requested count is 2, option 1.

## Key Points

- Java nested classes: static nested versus non-static inner; inner classes further include member, local, and anonymous forms.

## Why the other options are incorrect

Counts 3, 4, or 5 mix classification levels. If inner classes are expanded, they are commonly discussed as member inner, local inner, and anonymous inner classes; together with static nested classes this gives four concrete forms, but still only two top-level categories.
