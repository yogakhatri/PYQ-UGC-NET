# Question 60

*UGC NET CS · 2020 Nov With Answers · Web Programming · HTML Image Maps*

In HTML, <map> tag is used for

- **1.** defining a path between two nodes in an image
- **2.** defining clickable regions in an image
- **3.** highlighting an area in an image
- **4.** defining the site-map of a web-site

> [!TIP]
> **Correct answer: 2. defining clickable regions in an image**

## Solution

The HTML <map> element defines an image map. It is associated with an image through the image's usemap attribute and contains <area> elements whose shapes and coordinates identify clickable regions linked to different destinations. Therefore it is used to define clickable regions in an image, option 2.

## Key Points

- <map> groups <area> hotspots for a client-side clickable image map.

## Why the other options are incorrect

It does not describe graph paths between image nodes, visually highlight arbitrary areas, or create a website navigation sitemap. The name ‘map’ refers specifically to mapping portions of one image to links.
