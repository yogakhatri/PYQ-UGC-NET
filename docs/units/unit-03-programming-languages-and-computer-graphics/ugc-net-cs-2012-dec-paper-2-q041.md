# Question 41

*UGC NET CS · 2012 Dec Paper 2 · Elementary Data Types · Enumeration Types*

Enumeration is a process of:

- **1.** Declaring a set of numbers
- **2.** Sorting a list of strings
- **3.** Assigning the legal values possible for a variable
- **4.** Sequencing a list of operators

> [!TIP]
> **Correct answer: 3. Assigning the legal values possible for a variable**

## Solution

An enumeration defines a finite set of named legal values for a variable. For example, an enum Day might permit only MON, TUE, WED and the other named days. The names improve readability while the language assigns or represents them with underlying integral values.

## Key Points

- An enumerated type lists the complete set of named values that variables of that type may take.

## Why the other options are incorrect

An enumeration is not merely a declaration of arbitrary numbers, does not sort strings, and does not specify an execution sequence for operators. Its defining purpose is to restrict a value to one member of a declared named set.
