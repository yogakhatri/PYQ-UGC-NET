# Question 97

*UGC NET CS · 2023 Mar 11 Shift 2 Dec 2022 Session · Web Programming · HTML*

1: 137047 Which of the following lists defined in HTML share similar elements? A. Unordered lists B. Ordered lists C. Definition Lists D. Corelated lists E. Enumerated lists Choose the most appropriate answer from the options given below:

- **1.** A, B, D only
- **2.** A. C, E only
- **3.** A. B. C only
- **4.** A, C, D only

> [!TIP]
> **Correct answer: 3. A. B. C only**

## Solution

HTML defines three standard list structures: unordered lists (`ul`), ordered lists (`ol`), and description/definition lists (`dl`). These correspond to A, B, and C.

## Key Points

- HTML list containers are ul, ol, and dl.

## Why the other options are incorrect

‘Correlated list’ and ‘enumerated list’ are not separate HTML list elements; enumeration is handled by an ordered list.
