# Question 64

*UGC NET CS · 2013 Dec Paper 3 · 2-D Geometrical Transforms and Viewing · Line Half-Plane Test*

Which of the following points lies on the same side as the origin, with reference to the line 3x + 7y = 2 ?

- **A.** (3, 0)
- **B.** (1, 0)
- **C.** (0.5, 0.5)
- **D.** (0.5, 0)

> [!TIP]
> **Correct answer: D. (0.5, 0)**

## Solution

Put the line in signed form F(x,y)=3x+7y-2. Points are on the same side when their F values have the same sign. At the origin, F(0,0)=-2, which is negative. For (0.5,0), F=3(0.5)+7(0)-2=-0.5, also negative, so this point is on the origin's side.

## Key Points

- For ax+by+c=0, evaluate the left side: equal nonzero signs mean the same half-plane.

## Why the other options are incorrect

F(3,0)=7, F(1,0)=1 and F(0.5,0.5)=3; all three are positive, opposite to the origin's negative value. None lies on the line because none gives F=0.
