# Question 46

*UGC NET CS · 2015 Dec Paper 3 · Programming-Language Paradigms · Functional Programming and LISP*

Language model used in LISP is __________.

- **1.** Functional programming
- **2.** Logic programming
- **3.** Object oriented programming
- **4.** All of the above

> [!TIP]
> **Correct answer: 1. Functional programming**

## Solution

LISP evaluates programs primarily as expressions that compute and return values. Functions are first-class objects, recursion is a natural control mechanism, and list processing is expressed through function application. It is therefore classically categorized under the functional programming model.

## Key Points

- LISP is the classic functional/list-processing language; Prolog is the classic logic-programming language.

## Why the other options are incorrect

Logic programming is represented by languages such as Prolog, where programs are relations and rules solved by inference. Object-oriented programming organizes computation around encapsulated objects and message or method calls. Some Lisp dialects later acquired object systems and other extensions, but that does not change the paradigm asked for here, so 'all' is not the intended answer.
