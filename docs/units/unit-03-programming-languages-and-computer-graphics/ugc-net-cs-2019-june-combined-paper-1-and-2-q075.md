# Question 75

*UGC NET CS · 2019 June Paper 1 And 2 · Web Programming · JavaScript typing and HTML images*

Which of the following statements is/are true? P: In a scripting language like JavaScript, types are typically associated with values, not variables. Q: It is not possible to show images on a web page without the <img> tag of HTML. Select the correct answer.

- **1.** P only
- **2.** Q only
- **3.** Both P and Q
- **4.** Neither P nor Q

> [!TIP]
> **Correct answer: 1. P only**

## Solution

Statement P is true: JavaScript is dynamically typed, so a variable can hold values of different types and the type belongs to the current value. Statement Q is false: an image can be displayed without an HTML img element, for example as a CSS background image, inside SVG, or by drawing it on a canvas. Therefore only P is true.

## Key Points

- JavaScript variables are dynamically typed, and web images are not limited to the img element.

## Why the other options are incorrect

- **Option 2:** accepts the false image-tag claim and rejects the true typing claim.
- **Option 3:** incorrectly accepts Q.
- **Option 4:** incorrectly rejects P.
