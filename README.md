# UGC NET Computer Science: 45-Day Question Guide

This project turns previous-year questions into short, learner-focused solution
cards: the correct answer, a clear explanation, key points and why the other
options are incorrect.

## Start studying

- [45-day study plan](docs/45-day-plan.md)
- [Unit and subtopic index](docs/README.md)

Questions are solved independently from the questions-only corpus. Legacy
answer keys and unfinished answer inventories are not part of the guide.

## Build and check

```bash
python3 scripts/build_guide.py
python3 scripts/check_guide.py
```

The learner-facing guide contains only completed solution cards. Unsolved or
unverified answers remain hidden until they meet the same format and quality
standard.
